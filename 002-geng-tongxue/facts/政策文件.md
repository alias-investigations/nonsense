### 科技部令第24号《科学技术活动违规行为调查处理规定》

| 项目 | 内容 |
|------|------|
| 审议通过 | 2026年1月28日（科技部第1次部务会议） |
| 成文日期 | 2026年2月6日 |
| 公布日期 | 2026年2月11日 |
| **施行日期** | **2026年3月20日** |
| 前身 | 科技部令第19号（同日废止） |
| 来源 | 科技部官网、光明网、新浪财经 |

### 令第24号与令第19号的核心差异

| 对比项 | 令第19号（2020） | 令第24号（2026） |
|--------|-----------------|-----------------|
| 性质 | 暂行规定 | 正式规定（法律位阶提升） |
| 违规行为数 | 64种 | 68种 |
| 调查处理程序 | 无专章规定 | 增设"调查处理程序"专章 |
| 调查期限 | 无明确条文 | 明确自受理之日起6个月内完成（第34条） |
| 延期规定 | 无明确条文 | 经批准可延6个月，需向交办机关报告（第34条） |

### 令第24号调查期限原文

> **第三十四条** 科学技术活动违规行为的调查处理应当自决定受理之日起6个月内完成。
> 因特别重大复杂在前款规定期限内仍不能完成调查的，经主管部门有关负责人批准后可以延长调查期限，但延长不得超过6个月。对上级机关交办或其他执法执纪部门移送的，调查延期情况应当向交办机关或移送部门报告。

来源：科技部官网令第24号全文

## 四、教育部文件

### 教育部《关于加强高等学校科研诚信建设和学术不端治理的指导意见》

| 项目 | 内容 |
|------|------|
| 发文字号 | 教科信〔2024〕2号 |
| 成文日期 | 2024年3月26日 |
| 发布机构 | 教育部 |
| 来源 | 教育部官网、各高校科技处存档 |

核心内容：
- 严格落实科研诚信建设主体责任——高校是学术不端行为预防与处理的主体
- 党政主要负责人是第一责任人
- 严格执行"三落实、三公开"要求：公开学术不端行为调查处理结果，必须长期保留
- 落实约谈问责和通报机制

注：该文件于2024年发布。2026年5月20日左右多所高校系统转发执行，并非文件本身发布时间。

### 教育部《高等学校学术不端行为调查处理实施细则》

| 项目 | 内容 |
|------|------|
| 发文字号 | 教科信〔2024〕3号 |
| 发布机构 | 教育部 |
| 来源 | 教育部官网、各高校科技处存档 |

核心内容：
- 学术不端行为的调查处理应自决定受理之日起6个月内完成（第十一条）
- 处理决定由中央部门所属高校作出的，应在生效后10个工作日内将处理决定书和调查报告报送主管部门（第二十五条）

## 五、卫健委文件

### 《加强医学科研诚信专项治理的工作方案》

| 项目 | 内容 |
|------|------|
| 印发时间 | 2026年2月 |
| 发布机构 | 国家卫生健康委员会 |
| 来源 | 卫健委官网 |

### 执法行动

**2026年4月9日国家卫健委通报10起科研失信行为：**
- 涉及天津、辽宁、江苏、山东等7省
- 共处理41人（其中35名医护人员被处理，含副高职称撤职）
- 违规类型：代写代投、买卖论文、编造研究过程、伪造研究成果、篡改实验数据、无实质学术贡献署名
- 处罚措施：撤销职称、取消课题申报资格、追回科研经费、记入科研诚信严重失信数据库
- 依据文件：《加强医学科研诚信专项治理的工作方案》
- 来源：央广网、央视新闻、新浪财经
