# 中国法律体系基础

## 一、法律文件层级

```
        ┌─────────────┐
        │    宪法      │  最高效力
        ├─────────────┤
        │   法律       │  全国人大及其常委会制定
        │ (刑法、行政处 │  如：《刑法》《网络安全法》
        │  罚法等)      │
        ├─────────────┤
        │  行政法规     │  国务院制定
        │ (条例、规定)  │  如：《暂行规定》《电信条例》
        ├─────────────┤
        │  部门规章     │  国务院各部委制定
        │ (办法、通知)  │  如：《实施办法》《清理通知》
        ├─────────────┤
        │  司法解释     │  最高法、最高检制定
        │              │  如：法释[2000]12号
        ├─────────────┤
        │  规范性文件   │  各级政府、部门发布
        │ (意见、批复)  │  不属于严格意义的"法"
        └─────────────┘
```

**关键规则**：下位法不得与上位法抵触。如果抵触，上位法优先。

## 二、法律修订与实际适用

**核心原则：从新原则。**

实际判决依据的是**行为发生时有效的法律版本**，不是制定时的版本，也不是判决时的版本。

### 翻墙相关的法律修订历程

| 时间 | 事件 | 实质影响 |
|------|------|---------|
| 1996.02 | 暂行规定发布 | 第六条写"邮电部国家公用电信网" |
| 1997.05 | 第一次修订 | 修改具体措辞，核心条款不变 |
| 2024.03 | 第二次修订 | 仅删"邮电部"，改为"国家公用电信网" |

**吊诡之处**：1996年制定的、针对当时互联网基础设施状况的"暂行规定"，在互联网已经面目全非的今天，仍然是处罚翻墙的主要依据。核心条款从未实质性修改。

## 三、执法和司法差异

### 3.1 理论上：全国统一适用

宪法和立法法确立了法制统一原则。同一个法条，在北京和在贵州应该有同样的含义和适用。

### 3.2 现实中：差异显著

**执法层面（公安机关）：**

| 差异维度 | 表现 |
|---------|------|
| 定性不同 | 同样是翻墙，A地用第六条（非法定信道），B地用第十条（接入网络），C地用《网络安全法》第27条 |
| 处罚力度不同 | 同样是翻墙浏览，有的警告，有的罚款500元，有的罚款5000元 |
| 执法意愿不同 | 一线城市几乎不主动查处，三四线城市偶发性执法 |
| 发现方式不同 | 有的是技术手段发现，有的是检查手机发现，有的是举报 |

**司法层面（法院）：**

| 差异维度 | 表现 |
|---------|------|
| 罪名认定不同 | 同样是销售VPN，合肥判非法经营罪（9个月），上海宝山判提供工具罪（3年缓3年） |
| 是否入罪不同 | 同样是搭建VPN售卖，有的判实刑，有的判缓刑，有的不起诉 |
| 说理程度不同 | 大多数判决"简单罗列法条，没有释法说理" |

### 3.3 差异产生的原因

1. **法条本身的模糊性**："其他信道"没有定义，"接入网络"的含义有争议，VPN在法律中没有明确的定性
2. **缺乏有权解释**：全国人大常委会没有对暂行规定做过立法解释，国务院没有出过行政解释，最高法/最高检没有出过专门的司法解释
3. **执法政策的地区差异**：公安机关有较大的自由裁量空间，地方政府的态度影响执法力度

### 3.4 关键概念：指导性案例

最高法和最高检会发布"指导性案例"，对全国法院有"应当参照"的效力。**目前没有关于翻墙的指导性案例**。这意味着各地法院没有统一的裁判标杆。

## 四、对本研究的意义

1. 研究翻墙法律问题，必须同时关注**静态法条**和**动态执法**
2. 不能只看一个地方的案例就下结论，需要**跨地区比较**
3. 学术文献的价值在于提供**规范解释的框架**，帮助判断执法是否正确
4. 需要追踪**立法动态**，因为法律可能随时变化
