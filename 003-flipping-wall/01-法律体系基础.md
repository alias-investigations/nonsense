# 中国法律体系基础

## 一、法律文件层级

```
        ┌─────────────┐
        │    宪法      │  最高效力
        ├─────────────┤
        │   法律       │  全国人大及其常委会制定
        │ (刑法、行政处 │  如：《刑法》《网络安全法》
        │  罚法等)      │
        ├─────────────┤
        │  行政法规     │  国务院制定
        │ (条例、规定)  │  如：《暂行规定》《电信条例》
        ├─────────────┤
        │  部门规章     │  国务院各部委制定
        │ (办法、通知)  │  如：《实施办法》《清理通知》
        ├─────────────┤
        │  司法解释     │  最高法、最高检制定
        │              │  如：法释[2000]12号
        ├─────────────┤
        │  规范性文件   │  各级政府、部门发布
        │ (意见、批复)  │  不属于严格意义的"法"
        └─────────────┘
```

**关键规则**：下位法不得与上位法抵触。如果抵触，上位法优先。

## 二、法律修订与实际适用

**核心原则：从新原则。**

实际判决依据的是**行为发生时有效的法律版本**，不是制定时的版本，也不是判决时的版本。

### 翻墙相关的法律修订历程

| 时间 | 事件 | 实质影响 |
|------|------|---------|
| 1996.02 | 暂行规定发布 | 第六条写"邮电部国家公用电信网" |
| 1997.05 | 第一次修订 | 修改具体措辞，核心条款不变 |
| 2024.03 | 第二次修订 | 仅删"邮电部"，改为"国家公用电信网" |

**吊诡之处**：1996年制定的、针对当时互联网基础设施状况的"暂行规定"，在互联网已经面目全非的今天，仍然是处罚翻墙的主要依据。核心条款从未实质性修改。

## 三、执法和司法差异

### 3.1 理论上：全国统一适用

宪法和立法法确立了法制统一原则。同一个法条，在北京和在贵州应该有同样的含义和适用。

### 3.2 现实中：差异显著

**执法层面（公安机关）：**

| 差异维度 | 表现 |
|---------|------|
| 定性不同 | 同样是翻墙，A地用第六条（非法定信道），B地用第十条（接入网络），C地用《网络安全法》第27条 |
| 处罚力度不同 | 同样是翻墙浏览，有的警告，有的罚款500元，有的罚款5000元 |
| 执法意愿不同 | 一线城市几乎不主动查处，三四线城市偶发性执法 |
| 发现方式不同 | 有的是技术手段发现，有的是检查手机发现，有的是举报 |

**司法层面（法院）：**

| 差异维度 | 表现 |
|---------|------|
| 罪名认定不同 | 同样是销售VPN，合肥判非法经营罪（9个月），上海宝山判提供工具罪（3年缓3年） |
| 是否入罪不同 | 同样是搭建VPN售卖，有的判实刑，有的判缓刑，有的不起诉 |
| 说理程度不同 | 大多数判决"简单罗列法条，没有释法说理" |

### 3.3 差异产生的原因

1. **法条本身的模糊性**："其他信道"没有定义，"接入网络"的含义有争议，VPN在法律中没有明确的定性
2. **缺乏有权解释**：全国人大常委会没有对暂行规定做过立法解释，国务院没有出过行政解释，最高法/最高检没有出过专门的司法解释
3. **执法政策的地区差异**：公安机关有较大的自由裁量空间，地方政府的态度影响执法力度

### 3.4 关键概念：指导性案例

最高法和最高检会发布"指导性案例"，对全国法院有"应当参照"的效力。**目前没有关于翻墙的指导性案例**。这意味着各地法院没有统一的裁判标杆。

## 四、对本研究的意义

1. 研究翻墙法律问题，必须同时关注**静态法条**和**动态执法**
2. 不能只看一个地方的案例就下结论，需要**跨地区比较**
3. 学术文献的价值在于提供**规范解释的框架**，帮助判断执法是否正确
4. 需要追踪**立法动态**，因为法律可能随时变化

## 五、暂行规定的合宪性问题

### 5.1 行政法规制定程序条例的要求

**《行政法规制定程序条例》（2026年5月15日第二次修订）**

- **征求意见稿第6条**："行政法规应当备而不繁，逻辑严密，条文明确、具体，用语准确、简洁，具有可操作性，**符合立法技术规范**。"
- **正式版本第6条**："制定行政法规，应当坚持科学立法、民主立法、依法立法，增强立法的系统性、整体性、协同性、时效性。"
- **正式版本第25条**："国务院法治部门认为送审稿中涉及**合宪性问题**的，应当征求**全国人民代表大会常务委员会有关工作机构**意见。"

**暂行规定的问题**：
- 未定义"其他信道"和"国际出入口信道"两个核心概念
- 不符合"条文明确、具体"的要求
- 不符合"具有可操作性"的要求

### 5.2 宪法第40条的法律保留原则

**宪法第40条**：

> "中华人民共和国公民的通信自由和通信秘密受法律的保护。除因国家安全或者追查刑事犯罪的需要，由公安机关或者检察机关**依照法律规定的程序**对通信进行检查外，任何组织或者个人不得以任何理由侵犯公民的通信自由和通信秘密。"

**关键分析**：

| 事实 | 法律后果 |
|------|----------|
| 暂行规定限制了"国际联网" | 限制了公民的通信自由 |
| 暂行规定是行政法规，不是法律 | 不符合宪法第40条"依照法律规定的程序" |
| 宪法第40条要求"法律保留" | 限制通信自由必须由法律规定 |

**张翔教授的宪法学分析（2021+2023）**：

1. **通信权的保护范围**：通话记录当然落入宪法第40条通信权的保护范围（《通信权的宪法释义与审查框架》，2021年第1期《比较法研究》）

2. **"检查通信"是示例性规定**：如果将"检查通信"理解为"示例性规定"，则宪法第40条容有对通信权限制的其他可能性——但仍需满足法律保留要求

3. **通信权的双重法律保留结构**：
   - **加重法律保留**：第40条第2句——"除因国家安全或者追查刑事犯罪的需要，由公安机关或者检察机关依照法律规定的程序对通信进行检查外，任何组织或者个人不得以任何理由侵犯"——仅适用于"通信检查"
   - **单纯法律保留**：第40条第1句——"通信自由和通信秘密受法律的保护"——适用于其他限制通信权的方式

4. **法律保留的中国方案**（《基本权利限制法律保留的中国方案》，2023年《法律科学》）：
   - 中国宪法基本权利章的独特结构决定了不能照搬德国"单纯—加重—无"三层结构
   - 第51条排除了"无法律保留"阶层存在的可能性
   - "单纯—加重"的两层结构同时存在于人身自由、财产权、通信权等权利的**内部**

5. **对翻墙执法的意义**：
   - 限制通信自由必须由"法律"规定（全国人大及其常委会制定）
   - 行政法规（如暂行规定）不能限制通信自由
   - 这是宪法第40条的明确要求

### 5.3 学者背书

### 5.4 "暂行"27年的名称问题

**《行政法规制定程序条例》第39条**：

> "行政法规的名称一般称'条例'，也可以称'规定'、'办法'等。**国务院根据全国人民代表大会及其常务委员会的授权决定制定的行政法规，称'暂行条例'或者'暂行规定'。**"

**这意味着**："暂行"名称应该用于**授权立法**——即全国人大授权国务院制定的行政法规。

**但暂行规定不属于授权立法**：

| 事项 | 是否属于法律保留 |
|------|------------------|
| 犯罪和刑罚 | ✅ 是 |
| 限制人身自由的强制措施和处罚 | ✅ 是 |
| 网络管理 | ❌ 否 |

暂行规定涉及的是网络管理事项，不属于立法法第8条规定的"法律保留"事项。国务院可以**自行制定**行政法规，不需要全国人大授权。

**结论**：

| 事实 | 含义 |
|------|------|
| 暂行规定用了"暂行"名称 | 但**不属于授权立法** |
| 行政法规制定程序条例要求"暂行"用于授权立法 | 暂行规定的**名称不符合要求** |
| 暂行规定制定于1996年，早于立法法 | 当时没有明确的"暂行"使用规范 |

### 5.5 这是一个空白

虽然王全璋律师、张翔教授、全国人大常委会法工委等都论证了宪法第40条的"法律保留"原则，但**没有人直接论证暂行规定第6条违反宪法第40条**。

这是一个值得填补的空白。
