# 06 核心争议：五大争议深度分析

## 争议一：VPN是否属于"其他信道"？

### 问题
暂行规定第6条禁止"使用其他信道进行国际联网"，VPN是否属于此处的"其他信道"？

### 正方论据（支持处罚）
1. VPN实现了国际联网，且不是通过国家公用电信网提供的"法定"信道
2. 使用VPN绕过了GFW的审查，属于"非法定"的联网方式
3. 从文义解释看，"其他信道"包括一切非法定的联网信道

### 反方论据（反对处罚）
1. **体系解释**：实施办法第3条第3款将"国际出入口信道"定义为"物理信道"。"其他信道"应与"国际出入口信道"同属物理层面，即"其他物理信道"。VPN是虚拟加密隧道，不是物理信道。
2. **技术事实**：VPN用户的数据仍然通过运营商的物理信道（光缆）传输，并未建立独立的物理链路。VPN只是在应用层加密数据，并未改变物理层传输路径。
3. **目的解释**：1996年立法时，"其他信道"指的是私拉海底光缆、卫星天线等物理接入方式，与VPN技术无关。
4. **OSI模型论证**：物理信道属于"物理层"，VPN发生在"会话层"，属于两个不同层面。

### 律师界共识
**VPN不属于"其他信道"**。此观点得到王宇扬、周立波、陈蓟、程俊峰、肖飒等律师的一致支持。

### 学术界状态
**该问题在学术期刊上几乎没有专门讨论。** CSSCI/核心期刊论文主要讨论刑法问题（提供工具罪、非法经营罪），而非行政法上的"其他信道"解释问题。这是一个被学术界忽视的问题。

### 司法实践
**法院完全回避了这个问题。**

| 环节 | 状态 |
|------|------|
| 行政处罚决定书 | 直接定性"使用VPN=使用非法定信道"，不进行法理论证 |
| 刑事判决书 | 绕开"信道"概念，从"提供侵入工具"角度定罪 |
| 行政诉讼（赖亮平案） | 判决书未公开，无法确认是否讨论 |
| 律师辩护观点 | 法院不采纳，也不否定，完全回避 |

**结论：律师界的法理共识在实践中归零。**

### 评价
正方论据缺乏法律解释方法论的支持，属于望文生义。反方论据有体系解释、技术事实和立法史的支持，更有说服力。但法院不回应这些论据，而是通过转换定罪角度来回避法律争议。

## 争议二：翻墙是否违反"必须通过接入网络"？

### 问题
暂行规定第10条要求用户"必须通过接入网络进行国际联网"，翻墙是否违反此规定？

### 正方论据（支持处罚）
1. 翻墙用户没有通过接入网络访问境外网站
2. 翻墙绕过了接入网络的正常路径

### 反方论据（反对处罚）
1. 翻墙用户已经通过接入网络（ISP）接入了互联网
2. "入网"是从"不能上网"到"可以上网"，翻墙用户早已入网
3. 翻墙是在已入网的前提下访问被屏蔽的网站，不是"不通过接入网络"
4. "接入网络"在暂行规定第3条中有明确定义，指的是网络层级结构，而非访问特定网站的权限

### 学界共识
**翻墙不违反"必须通过接入网络"**。翻墙用户已经通过接入网络入网，翻墙只是在已入网的基础上访问了特定境外网站。

### 评价
正方论据混淆了"入网"和"访问特定网站"两个概念。反方论据更符合"接入网络"的法律定义。

## 争议三：提供翻墙工具是否构成犯罪？

### 问题
提供VPN/SSR等翻墙工具的行为，是否构成刑事犯罪？

### 可能涉及的罪名
1. **提供侵入、非法控制计算机信息系统程序、工具罪**（刑法第285条第3款）
2. **非法经营罪**（刑法第225条）
3. **拒不履行信息网络安全管理义务罪**（刑法第286条之一）

### 罪名一：提供侵入非法控制工具罪

**构成要件**：
- 提供专门用于侵入、非法控制计算机信息系统的程序、工具
- 或明知他人实施侵入、非法控制计算机信息系统的违法犯罪行为而为其提供程序、工具
- 情节严重

**不构成的理由**：
1. VPN不是"专门用于"侵入、非法控制计算机信息系统的工具——VPN具有广泛的合法用途（企业远程办公、高校访问学术资源等）
2. GFW不是本罪要保护的"计算机信息系统"——GFW是网络审查系统，不是他人的具体计算机信息系统
3. GFW不是"计算机信息系统安全保护措施"——GFW的作用是内容审查，不是保护数据安全
4. 翻墙没有"侵入"任何计算机信息系统——翻墙是加密通讯，不是未经授权的访问
5. 帮助行为独立化要求被帮助行为本身构成犯罪——翻墙本身不构成犯罪

### 罪名二：非法经营罪

**构成要件**：
- 违反国家规定
- 非法经营行为
- 扰乱市场秩序
- 情节严重

**不构成的理由**：
1. 翻墙服务不属于《电信业务目录》中的任何业务类型
2. 翻墙服务本身被禁止，不存在可以合法经营的市场
3. 本质上是违反网络审查秩序的电信活动，不是非法经营电信业务行为
4. 不符合非法经营罪"兜底条款"的适用条件——无法律、司法解释的明确规定

### 罪名三：拒不履行信息网络安全管理义务罪

**可能构成的条件**：
- 提供翻墙软件行为人，在公安、网信等部门发现有人使用翻墙软件传输违法犯罪信息
- 经责令停止联网后仍拒不停止
- 致使违法信息大量传播等情节严重

**评价**：特定条件下可能构成，但仍需满足"情节严重"的认定标准。

### 学界共识
**提供翻墙工具不构成提供侵入非法控制工具罪和非法经营罪**。此观点得到陈蓟、周立波、程俊峰、刘海等学者/律师的一致支持。

## 争议四：GFW的法律地位

### 问题
GFW在法律上是什么？是计算机信息系统？是安全保护措施？是技术措施？

### 观点A：GFW不是计算机信息系统
- GFW是网络审查系统的统称，不是他人的具体的计算机信息系统
- 该系统并未存在于任何官方文件中
- 网络审查系统本身的合法性和正当性存在疑问
- 法律没有明确将其上升为应该保护的法益

### 观点B：GFW不是计算机信息系统安全保护措施
- "计算机保护措施"一般指数据所有者出于数据安全考虑设立的措施
- GFW的作用是内容审查，不是保护数据安全
- 被屏蔽的境外网站数据不归属于我国境内任何主体
- GFW不是基于保护境外网站数据安全设立的

### 观点C：GFW是"技术措施"（数安条例第39条）

- 数安条例第39条将GFW定位为"技术措施"
- 破坏、避开该等技术措施的，构成行政违法（仅针对提供者）

**重要发现**：数安条例第39条是第一个在法规层面间接承认GFW存在的法律文件。

- 之前：政府从未正式承认GFW的存在
- 现在：数安条例第39条通过"技术措施"的表述，间接承认了GFW的存在
- 但这只是间接承认，不是直接承认

**"技术措施"的含义**：

1. **"国家采取措施"**：承认国家已经采取了技术措施，这些措施是为了"防范、处置网络数据跨境安全风险和威胁"
2. **"破坏、避开技术措施"**：禁止提供专门用于破坏、避开这些技术措施的工具（VPN就是用于"避开"这些技术措施的工具）
3. **仅禁止"提供"，不禁止"使用"**：个人使用VPN不违反数安条例第39条

**GFW被定位为"技术措施"的法律意义**：

- 不是"计算机信息系统"（刑法第285条保护的对象）
- 不是"安全保护措施"（刑法第285条保护的对象）
- 而是"技术措施"（数安条例第39条保护的对象）
- 这个定位有法律意义：破坏、避开"技术措施"的处罚比破坏"计算机信息系统"轻得多
- 不禁止个人"使用"

### 学界共识
**GFW不是计算机信息系统，也不是计算机信息系统安全保护措施**。此观点得到学界多数支持。数安条例将GFW定位为"技术措施"，而非"安全保护措施"。

## 争议五：翻墙行为的违法性定位

### 问题
翻墙行为本身是否违法？如果违法，是行政违法还是刑事犯罪？

### 观点A：行政违法（但不构成犯罪）——学界主流
- 翻墙行为本质上是违反网络审查秩序的行为
- 类似网络空间领域的"偷渡"
- 目前只涉嫌违反行政法规和行业规范
- 不构成犯罪

### 观点B：不违法
- 单纯使用VPN翻墙行为本身并未违反中国现行法律法规
- VPN不是自行建立信道进行国际联网
- 使用VPN仍然使用国家公用电信网提供的国际出入口信道
- VPN只是规避了GFW对某些境外IP的屏蔽

### 观点C：违法但处罚依据错误
- 翻墙行为可能违反暂行规定
- 但现有行政处罚的法律适用普遍存在错误
- 处罚机关均无权对翻墙行为实施行政处罚

### 学界共识
**翻墙行为最多构成行政违法，不构成犯罪**。但即使认定为行政违法，现有处罚的法律适用也普遍存在错误。

### 评价
此争议的核心不是"翻墙是否违法"，而是"如果违法，应当如何合法地规制"。目前的执法实践缺乏明确的法律依据和统一的执法标准。

## 争议六：翻墙后的收入是否属于"违法所得"？

### 问题
翻墙期间获得的全部收入（如程序员工资、虚拟货币交易收入），是否属于"违法所得"应予没收？

### 正方论据（支持没收）
1. 翻墙是违法行为，翻墙期间的一切收入都是"因违法行为取得的款项"
2. 暂行规定第14条明确规定"有违法所得的，没收违法所得"
3. 不没收违法所得不足以惩罚违法行为

### 反方论据（反对没收）

**法律依据**：
- 行政处罚法第28条："违法所得是指实施违法行为所取得的款项"
- 最高人民法院研究室意见：违法所得应是指获利数额，即全部收入扣除直接用于经营活动的合理支出后剩余的数额
- 工商行政管理机关行政处罚案件违法所得认定办法：以违法生产、销售商品或提供服务所获得的全部收入扣除直接用于经营活动的适当合理支出，为违法所得

**分析逻辑**：
1. **违法行为本身不产生收入**：翻墙本身并不能带来收入。带来收入的是程序员给外国公司提供的服务（编写代码、售后支持等）。
2. **服务本身不违法**：如果这个服务本身不是违法行为，这个服务的报酬就不应该算成违法所得。
3. **违法所得的正确计算**：即使用最严格的标准，也应计算使用非法VPN获得的**额外获益**（节省的成本），而非全部收入。
   - 合法外网专用信道专线：约2万元/年
   - 共享云服务合法外网访问：几百元/月
   - 三年额外收益（节省成本）：大几千到几万元
4. **因果关系**：程序员的收入是劳动报酬，不是翻墙的对价。翻墙是手段，编写代码是目的。手段的违法性不能自动传导到目的的合法性。

**数据支撑**：
- 2021年涉违法所得行政案件：102件中，30%判决撤销或变更行政处罚决定
- 远高于一般行政诉讼败诉率

### 实践中的差异
| 案例 | 翻墙行为 | 收入来源 | 认定为违法所得 | 处理方式 |
|------|----------|----------|----------------|----------|
| 承德程序员案 | 翻墙远程工作 | 工资收入 | 105.8万全部收入 | 全部没收 |
| 炒币翻墙案 | 翻墙从事虚拟货币交易 | 交易收入 | 9万元 | 没收 |
| 张韬舟山案 | 翻墙访问维基百科 | 无 | 无 | 当场训诫 |

### 学界共识
**翻墙期间的全部收入不应被认定为违法所得**。违法所得必须与违法行为有直接因果关系。翻墙本身不产生收入，劳动报酬不是翻墙的对价。

### 评价
承德案的违法所得认定存在严重问题。按此逻辑，任何使用过VPN的人，其此后获得的所有收入都可以被没收，这显然超出了法理和常识的范畴。违法所得的认定需要建立统一标准，明确"违法行为产生的额外获益（利润）而非全部收入"的原则。

## 争议七：暂行规定是否符合行政法规制定程序？

### 问题
暂行规定作为行政法规，是否符合《行政法规制定程序条例》的要求？特别是：
1. 是否符合"条文明确、具体"的要求？
2. 是否符合"用语准确、简洁"的要求？
3. 是否符合"具有可操作性"的要求？

### 相关规定

**《行政法规制定程序条例》（2026年5月15日第二次修订）**

- **征求意见稿第6条**："行政法规应当备而不繁，逻辑严密，条文明确、具体，用语准确、简洁，具有可操作性，**符合立法技术规范**。"
- **正式版本第6条**："制定行政法规，应当坚持科学立法、民主立法、依法立法，增强立法的系统性、整体性、协同性、时效性。"
- **正式版本第25条**："国务院法治部门认为送审稿中涉及**合宪性问题**的，应当征求**全国人民代表大会常务委员会有关工作机构**意见。"

### 暂行规定的问题

**1. 未定义核心概念**

暂行规定未定义以下核心概念：
- "其他信道"：第6条禁止"使用其他信道进行国际联网"，但未定义何为"其他信道"
- "国际出入口信道"：第5条提到"通过接入网络进行国际联网"，但未定义何为"国际出入口信道"

实施办法第3条第3款将"国际出入口信道"定义为"物理信道"，但这只是部门规章，法律层级低于暂行规定。

**2. 条文不明确、不具体**

- "其他信道"的含义不明确，导致执法部门可以任意解释
- "国际出入口信道"的定义不明确，导致VPN是否属于"其他信道"存在争议

**3. 缺乏可操作性**

- 由于核心概念未定义，执法部门在实践中只能依赖地方裁量基准
- 不同地方对同一行为的处罚差异巨大（警告 vs 5000元罚款）

### 分析

**1. 征求意见稿的要求**

征求意见稿明确要求行政法规"符合立法技术规范"。立法技术规范要求：
- 定义条款应当明确、具体
- 用语应当准确、简洁
- 条文应当具有可操作性

暂行规定未定义"其他信道"和"国际出入口信道"，不符合这些要求。

**2. 正式版本的要求**

正式版本强调"合宪性审查"。如果行政法规涉及合宪性问题，国务院应当征求全国人大常委会意见。

暂行规定第6条可能涉及宪法第40条（通信自由和通信秘密）的合宪性问题，但未见国务院征求全国人大常委会意见的记录。

**3. 对暂行规定的影响**

暂行规定可能不符合《行政法规制定程序条例》的要求：
- 未定义核心概念，不符合"条文明确、具体"的要求
- 缺乏可操作性，导致执法混乱
- 可能涉及合宪性问题，但未进行合宪性审查

### 学界观点

**立法技术规范的法律效力**：
- 立法法第53条授权法工委编制立法技术规范
- 但技术规范标注"试行"供参考
- 2024年正式版图书出版（无"试行"标注）
- 法律效力不确定，但代表最佳实践

**暂行规定的立法缺陷**：
- 王宇扬、周立波、陈蓟等学者一致认为暂行规定存在立法缺陷
- 未定义核心概念是主要问题
- 但这不是"违法"，只是未遵循最佳实践

**暂行规定"暂行"了27年**：
- 行政法规制定程序条例第39条要求"暂行"名称用于**授权立法**
- 但暂行规定不属于授权立法（网络管理不属于法律保留事项）
- 国务院可以自行制定行政法规，不需要全国人大授权
- **名称与实质不符**：用了"暂行"名称，但不属于授权立法
- 陈寒非指出：暂行法"无明确的时间条款"，"缺乏明确有效的总结提升机制"
- 胡凌指出：国务院"似乎没有进一步修正的意向"

### 评价

暂行规定作为行政法规，应当符合《行政法规制定程序条例》的要求。但暂行规定未定义"其他信道"和"国际出入口信道"两个核心概念，不符合"条文明确、具体"的要求，也不符合"具有可操作性"的要求。

这导致了两个后果：
1. 执法部门可以任意解释"其他信道"，将VPN纳入禁止范围
2. 不同地方对同一行为的处罚差异巨大

## 争议八：暂行规定是否违反宪法第40条？

### 问题

暂行规定第6条限制公民"使用其他信道进行国际联网"，是否违反宪法第40条关于通信自由和通信秘密的保护？

### 宪法条文

**宪法第40条**：

> "中华人民共和国公民的通信自由和通信秘密受法律的保护。除因国家安全或者追查刑事犯罪的需要，由公安机关或者检察机关**依照法律规定的程序**对通信进行检查外，任何组织或者个人不得以任何理由侵犯公民的通信自由和通信秘密。"

### 关键分析

**1. "法律保留"原则**

宪法第40条使用了"依照法律规定的程序"这一表述。在宪法学中，"法律"有特定含义——指全国人大及其常委会制定的法律，不包括行政法规。

**2. 暂行规定的层级**

暂行规定是行政法规（国务院制定），不是法律（全国人大及其常委会制定）。

**3. 这意味着什么**

| 事实 | 法律后果 |
|------|----------|
| 暂行规定限制了"国际联网" | 限制了公民的通信自由 |
| 暂行规定是行政法规，不是法律 | 不符合宪法第40条"依照法律规定的程序" |
| 宪法第40条要求"法律保留" | 限制通信自由必须由法律规定 |

### 学者背书

**1. 王全璋律师（2026年2月5日）**

已向全国人大常委会提交建议书，指公安部《网络犯罪防治法（征求意见稿）》"越权立法"，违反宪法和立法法。核心论点：

> "凡涉及剥夺公民政治权利、限制人身自由的强制性措施和处罚，必须由全国人民代表大会及其常委会通过立法程序制定（即制定法律），而不能由作为行政执法部门的公安部通过部门规章或行政法规来确立。"

来源：https://www.secretchina.com/news/b5/2026/02/06/1094629.html

**2. 张翔教授（中国人民大学法学院）**

明确论证：

> "按照我国宪法第40条，有权对通信进行检查的主体只能是公安机关或者检察机关……宪法第40条都确定无疑地将检查通信权规定为一项国家公权力。"

来源：http://law.newdu.com/c/20230820/4115.html

**3. 全国人大常委会法工委（2019年备案审查报告）**

指出：

> "有的地方性法规规定，公安机关交通管理部门调查交通事故时可以查阅、复制当事人通讯记录。经审查认为，该规定不符合保护公民通信自由和通信秘密的原则和精神；**对公民通信自由和通信秘密保护的例外只能是在特定情形下由法律作出规定**，有关地方性法规所作的规定已超越立法权限。"

来源：全国人大常委会法工委2019年备案审查工作报告

**4. 王宇扬律师**

系统梳理了50+翻墙处罚案例，论证翻墙行政处罚系适用法律错误。

来源：https://mp.weixin.qq.com/s/rFtHShfad2YouX84ar5SlQ

### 争议焦点

**1. 暂行规定第6条是否构成对通信自由的限制？**

- **正方**：暂行规定第6条禁止"使用其他信道进行国际联网"，限制了公民选择通信方式的自由
- **反方**：暂行规定限制的是"国际联网"的方式，不是"通信"本身

**2. "国际联网"是否属于"通信"？**

- **正方**：国际联网是现代通信的主要方式，限制国际联网就是限制通信
- **反方**：国际联网是网络管理概念，不是通信概念

**3. 宪法第40条的"法律"是否包括行政法规？**

- **正方**：宪法中的"法律"特指全国人大及其常委会制定的法律，不包括行政法规
- **反方**：宪法中的"法律"可以包括行政法规

### 现有案例

**1. 赖亮平案（2019年）**

赖亮平因翻墙被处罚后，向上杭县公安局提起行政诉讼。法院回避了暂行规定是否违反宪法第40条的问题，仅审查了处罚程序的合法性。

**2. 王全璋律师的建议书（2026年）**

王全璋律师向全国人大常委会提交建议书，指《网络犯罪防治法（征求意见稿）》违宪。但这是针对新法草案，不是针对暂行规定。

### 评价

**1. 这是一个尚未被公开论证的空白**

虽然王全璋律师、张翔教授、全国人大常委会法工委等都论证了宪法第40条的"法律保留"原则，但**没有人直接论证暂行规定第6条违反宪法第40条**。

这是一个值得填补的空白。

**2. 可能的论证路径**

- 暂行规定第6条限制了公民选择通信方式的自由
- 这种限制必须由"法律"规定
- 暂行规定是行政法规，不是法律
- 因此，暂行规定第6条可能违反宪法第40条

**3. 可能的反驳**

- "国际联网"不是"通信"
- 暂行规定不是限制通信，而是管理网络
- 宪法第40条的"法律"可以包括行政法规

**4. 这个论证的意义**

如果暂行规定第6条被认定为违反宪法第40条，那么：
- 所有依据暂行规定第6条作出的翻墙行政处罚都可能被撤销
- 国务院需要修订暂行规定，或者由全国人大常委会制定法律
- 翻墙行为的法律地位将发生根本变化

### 张翔教授的宪法学分析（2021+2023）

**通信权的宪法释义与审查框架**（《比较法研究》2021年第1期）：

1. **通信权的保护范围**：
   - 通话记录当然落入宪法第40条通信权的保护范围
   - 无需借助隐私权或个人信息权来保护

2. **"检查通信"是示例性规定**：
   - 如果将"检查通信"理解为"示例性规定"，则宪法第40条容有对通信权限制的其他可能性
   - 但这些可能性仍需满足法律保留要求

3. **通信权的分层构造**：
   - "通信内容"和"非内容的通信信息"应区分保护
   - 通信内容受到最严格保护（加重法律保留）
   - 非内容的通信信息受到相对宽松保护（但仍需法律保留）

**基本权利限制法律保留的中国方案**（《法律科学》2023年第6期）：

1. **中国宪法的独特结构**：
   - 不能照搬德国"单纯—加重—无"三层结构
   - 第51条排除了"无法律保留"阶层存在的可能性

2. **"单纯—加重"的两层结构**：
   - 同时存在于人身自由、财产权、通信权等权利的**内部**
   - 通信权：第40条第2句（通信检查）是加重法律保留；第40条第1句（通信自由和秘密）是单纯法律保留

3. **"示例性规定"的作用**：
   - 宪法基本权利条款中的限制规定具有"示例性规定"的性质
   - 不能简单地认为"只有示例性规定列举的限制方式才被允许"
   - 但也必须满足法律保留的基本要求

### 学界观点

**宪法学界的共识**：
- 宪法第40条确立了"法律保留"原则
- 限制通信自由必须由法律规定
- 行政法规不能限制通信自由

**但对暂行规定的适用**：
- 尚无学者直接论证暂行规定第6条违反宪法第40条
- 这可能是因为暂行规定制定于1996年，早于宪法学研究的繁荣期
- 也可能是因为翻墙执法是2018年后才大规模出现的，学界尚未跟进

## 争议九：律师辩护策略的有效性

### 问题

律师界达成的法理共识（VPN≠其他信道）在司法实践中完全无效，那么律师还能做什么？辩护策略还有什么意义？

### 辩护策略有效性排序

基于刘洋、董佳男等学者对50+案例的分析，辩护策略有效性从高到低：

| 排序 | 策略 | 有效性 | 说明 |
|------|------|--------|------|
| 1 | **处罚前与公安谈判** | 最有效 | 最常见，但最不透明 |
| 2 | **程序性辩护** | 较有效 | 追诉时效、程序违法 |
| 3 | **事实/证据抗辩** | 较有效 | 质疑VPN日志真实性、使用范围 |
| 4 | **法律适用错误** | 中等 | 公安引用了错误的法条 |
| 5 | **技术定性** | 中等 | VPN实际未翻墙（如IEPL专线） |
| 6 | **从轻情节** | 中等 | 初犯、无有害内容、配合 |
| 7 | **"其他信道=物理信道"** | **极低** | 法院系统性无视 |
| 8 | **"法规过时"** | **极低** | 法院不会质疑行政法规效力 |
| 9 | **"无明确禁止个人使用"** | **极低** | 法院认为"任何单位和个人"已涵盖 |
| 10 | **宪法/权利论证** | **极低** | 法院完全回避 |

### 法院实际回应 vs. 无视的辩护类型

**法院无视的（实质性法律论证）**：
- "其他信道=物理信道"
- "法规过时"
- "无明确禁止个人使用"
- "技术中立"

**法院回应的（程序/事实论证）**：
- 程序违法
- 证据不足
- 追诉时效
- 法律适用错误
- 从轻情节

### 成功案例

**1. 吴某涉销售VPN软件案（2019）——撤案（刑事案件）**
- **案件性质**：刑事案件，指控罪名是"提供侵入、非法控制计算机信息系统程序、工具罪"（刑法第285条第3款）
- **辩护重点**：缺乏主观明知、违法性认识不足、证据不足、当事人仅从事客服工作
- **结果**：检察院采纳辩护意见，退回公安，公安撤案
- **注意**：此案是刑事案件，法律依据是刑法第285条第3款，不是暂行规定第6条
- **来源**：https://www.163.com/dy/article/F1B7LPGO0525T0BA.html

**2. 中山市李某案（2024）——相对不起诉**
- 辩护：情节轻微、认罪认罚
- 结果：相对不起诉，但仍被行政拘留15天
- 关键：**从轻情节**
- 来源：https://zhongshan.jcy.gov.cn/network_security/202507/t20250702_627053.html

**3. 何思辰律师辩护案（2026）——成功**
- 辩护：技术定性+法律适用
- 结果：检察院采纳辩护意见
- 来源：https://www.66law.cn/laws/16446198.aspx

### 失败案例

**1. 浙江62起案例（2019-2020）**
- 结果：零成功辩护记录
- 所有案件均以警告或罚款结案

**2. 赖亮平案（2019）——唯一行政诉讼**
- 结果：判决书未公开，无法确认法院是否回应法理论证
- 意义：这是唯一进入行政诉讼的案件，但判决书未公开

### 为什么实质性法律辩护失败？

**1. 政治敏感性**
- 翻墙案件涉及国家网络政策，法院不愿挑战
- 法院面临支持公安执法的体制压力

**2. 成本收益不对等**
- 罚款金额低（通常警告或几百元）
- 法院不值得为这点钱挑战公安

**3. 结构性鸿沟**
- 律师把问题框定为"法规是否正确"
- 法院当作"执法是否正确"处理
- 法院系统性回避宪法/法规有效性问题

### 律师界共识的现实命运

**法理共识在实践中归零**：
- 法院不采纳，也不否定，完全回避
- 行政处罚决定书直接定性，不进行法理论证
- 刑事判决书绕开"信道"概念，从其他角度定罪

**但这不意味着法理辩护没有价值**：
- 它为学术界和立法讨论提供了素材
- 它为未来可能的法治进步留下了记录
- 它向同行和公众传递了"这个问题有法律争议"的信号

### 核心结论

**1. 辩护策略的现实选择**

如果律师想在翻墙案件中取得实际效果，应当：
- **优先选择程序性辩护**（追诉时效、程序违法）
- **重点质疑证据质量**（VPN日志真实性、使用范围）
- **避免实质性法律论证**（"其他信道=物理信道"）
- **争取从轻情节**（初犯、无有害内容、配合）

**2. 法理辩护的意义**

虽然法理辩护在实践中无效，但它仍然有意义：
- 为学术讨论提供素材
- 为未来立法改革留下记录
- 向公众传递"这个问题有争议"的信号

**3. 核心问题**

不是"律师能做什么"，而是"**在Rule by Law体系下，律师的角色是什么？**"

- 如果法院不讲法理，律师的法理辩护还有什么意义？
- 律师是在为当事人辩护，还是在为"法治"辩护？
- 如果答案是后者，那么这些文章就不是写给法院看的，而是写给未来看的。

## 争议十：法律文本与实际执行的脱钩

### 问题

暂行规定第6条写的是"其他信道"，但实际执法和司法中使用的是"翻墙"。这两者之间是否存在脱钩？

### 核心证据

**赖亮平案（2019年行政判决书）**——最直接的证据：

> "被告上杭县公安局作出的杭公（庐丰）行罚决字〔2019〕00031号行政处罚决定，系对赖亮平违反《中华人民共和国计算机信息网络国际联网管理暂行规定》第六条第二款、第十四条的规定，即**'翻墙'访问国外网站的违法行为**处以停止联网、警告的行政处罚"

**注意这个表述**：
- 法律文本：暂行规定第6条说的是"其他信道"
- 法院判决：直接写"翻墙访问国外网站"
- 法院把"翻墙"等同于"违反第6条"

### 更多案例

| 案例 | 法律依据 | 文书中的表述 |
|------|----------|-------------|
| 朱某案（广东南雄，2018） | 暂行规定第6条 | "安装翻墙软件蓝灯...翻墙上网" |
| 赖亮平案（福建上杭，2019） | 暂行规定第6条 | "翻墙访问国外网站的违法行为" |
| 陈圣哲案（浙江温州，2020） | 暂行规定第6条 | "使用非法路由器'翻墙'登录搜索" |
| 潘炳霖案（浙江岱山，2020） | 暂行规定第6条 | "下载了'UU加速器'翻墙软件" |
| 陈诚案（浙江衢州，2020） | 暂行规定第6条 | "下载一款叫'Shadowrocket'的翻墙软件" |
| 杨某案（石家庄元氏，2023） | 暂行规定第6条 | "使用'翻墙软件'进行国际联网" |

### 法律文本与实际执行的脱钩

| 层面 | 使用的概念 |
|------|-----------|
| 暂行规定第6条（法律文本） | "其他信道" |
| 行政处罚决定书 | "翻墙" |
| 行政判决书 | "翻墙" |

**法院直接把"翻墙"等同于"违反第6条"**，而没有论证"翻墙"是否属于"其他信道"。

### 深层含义

**1. "翻墙"已经成为独立的法律概念**

虽然暂行规定第6条没有使用"翻墙"这个词，但在实际执法和司法中，"翻墙"已经成为一个独立的、被广泛接受的法律概念。

**2. 法律文本的"信道"概念被架空**

暂行规定第6条的"其他信道"在实际执行中被"翻墙"取代。法律文本的"信道"概念被架空了。

**3. 法院没有进行法律解释**

法院直接把"翻墙"等同于"违反第6条"，而没有论证：
- "翻墙"是否属于"其他信道"
- "翻墙"与"其他信道"的关系是什么
- 为什么"翻墙"可以等同于"违反第6条"

**4. 这是Rule by Law的典型表现**

法律文本写的是"其他信道"，但实际执行的是"翻墙"。法律的字面含义不重要，重要的是执法机关如何认定。

### 对辩护策略的影响

如果法院直接把"翻墙"等同于"违反第6条"，那么：

**无效的辩护**：
- "VPN不是其他信道"（回应的是法律文本，不是法院的实际认定）
- "其他信道指的是物理信道"（回应的是法律理论，不是法院的实际认定）

**有效的辩护**：
- "未翻墙"（直接回应法院的实际认定）
- "翻墙行为未被发现/证据不足"（质疑事实认定）
- "程序违法"（质疑执法程序）

### 核心结论

**律师界的理论辩论打错了靶子。**

律师们在辩论"VPN是否属于其他信道"，但法院直接把"翻墙"等同于"违反第6条"，根本不讨论"其他信道"。

这意味着：
1. 法律文本（"其他信道"）与实际执行（"翻墙"）已经脱钩
2. 律师的理论辩护需要重新定位——回应法院的实际认定，而不是法律文本
3. "翻墙"已经成为一个独立的、被广泛接受的法律概念，尽管暂行规定没有使用这个词

### 关键发现

**1. 北京市是最明确的**

北京市2025年版明确提及《暂行规定》第六条、第八条、第十条，并设定了三级罚款标准：
- 一级：警告 + 个人罚款2000元以下，单位5000元以下
- 二级：警告 + 个人罚款2000-10000元，单位5000-10000元
- 三级：警告 + 个人罚款10000-15000元，单位10000-15000元

来源：https://www.beijing.gov.cn/zhengce/zhengcefagui/202512/t20251202_4316838.html

**2. 大多数省份不明确提及VPN/翻墙**

它们引用的是一般性法律条款（如"计算机和网络安全"），而不是专门针对VPN/翻墙的条款。

**3. 没有全国统一标准**

每个省根据自己的情况制定裁量基准，导致执法不统一。

### 与翻墙案件的关联

| 案例 | 省份 | 裁量基准 | 处罚结果 |
|------|------|----------|----------|
| 韶关朱云枫案 | 广东 | 广东省出入境管理裁量基准 | 警告 + 1000元 |

### 核心结论

**1. 执法依据不统一**

- 有的省份引用《暂行规定》第六条（如北京）
- 有的省份引用《网络安全法》第二十七条（如上海）
- 有的省份引用一般性条款（如浙江）

**2. 裁量基准不明确提及VPN/翻墙**

大多数省级裁量基准不明确提及VPN/翻墙，而是引用一般性法律条款。这导致执法具有很大的自由裁量空间。

**3. 没有全国统一标准**

每个省根据自己的情况制定裁量基准，导致执法不统一。例如：
- 北京：三级罚款标准（2000/10000/15000元）
- 上海：详细分级
- 浙江：62起案件，大多警告

**4. 这是翻墙执法混乱的制度性原因**

- 不同省份引用不同的法律条款
- 不同省份的裁量基准不同
- 同样的行为在不同省份可能面临不同的处罚

## 争议十二：法律与执法的gap反映了什么？

### 问题

法律写的是"其他信道"，执法用的是"翻墙"，这个gap反映了什么？

### gap的表现

| 层面 | 使用的概念 |
|------|-----------|
| 暂行规定第6条（法律文本） | "其他信道" |
| 行政处罚决定书 | "翻墙" |
| 行政判决书 | "翻墙" |

**法院直接把"翻墙"等同于"违反第6条"**，而没有论证"翻墙"是否属于"其他信道"。

### 这个gap是故意的

**政府不明确禁止VPN，不是因为疏忽，而是因为政治计算。**

### 三层逻辑

**1. 法律层：故意模糊**

- 暂行规定第6条："其他信道"——未定义
- 实施办法第3条：定义"国际出入口信道"为"物理信道"——但法律层级低
- 没有任何法律明确禁止VPN

**2. 执法层：选择性适用**

- 公安引用暂行规定第6条
- 将"其他信道"解释为包括VPN
- 选择性执法：处罚极少数人，威慑大多数人

**3. 政治层：维护模糊**

- 明确禁止VPN = 承认网络审查
- 明确禁止VPN = 宪法争议
- 明确禁止VPN = 国际形象问题
- 保持模糊 = 最优策略

### 这反映了什么

**1. Rule by Law的本质**

- 法律不是约束政府的规则
- 法律是政治控制的工具
- 法律的含义随政治需要变化
- 暂行规定第6条的"其他信道"可以被解释为任何东西

**2. 选择性执法是特征，不是bug**

- 翻墙用户数以千万计
- 被处罚者极少
- 这不是执法不力，而是故意选择
- 选择性执法可以：
  - 震慑特定人群
  - 避免大规模执法成本
  - 保持政治灵活性
  - 避免国际形象问题

**3. 模糊治理的逻辑**

- 法律不明确禁止
- 但执法机关可以任意解释
- 处罚轻，成本低，没人追究
- 学术争议无法进入司法程序
- 维持现状对政府最有利

**4. 宪法被架空**

- 宪法第35条：言论自由
- 宪法第40条：通信自由和通信秘密
- 但没有法律明确授权限制这些权利
- GFW和翻墙执法都建立在宪法灰色地带

### 核心结论

**执法与法律的gap不是法律体系的失败，而是Rule by Law的成功。**

- 模糊的法律给了执法机关最大的灵活性
- 选择性执法可以针对特定人群
- 避免了宪法争议和国际形象问题
- 维持了政治控制的有效性

**这个gap本身就是政治工具。**

## 争议十三：为什么没有法律明确禁止VPN？

### 问题

既然公安、国安认为翻墙违法，为什么没有任何现行法律明确这一点？2026年新订立的法律为什么也要回避？

### 原因分析

**1. 政治原因**

- 明确禁止VPN = 承认网络审查
- 国际形象问题
- 人权问题
- 外交压力

**2. 法律原因**

- 宪法第35条：言论自由
- 宪法第40条：通信自由和通信秘密
- 明确禁止VPN可能违宪

**3. 经济原因**

- 企业需要VPN（远程办公、跨境业务）
- 研究人员需要VPN（学术访问）
- 全面禁止VPN会损害经济
- 选择性执法更灵活

**4. 技术原因**

- GFW本身就是技术控制
- 不需要法律明确禁止
- 技术手段比法律手段更有效

### 2026年新法律为什么也要回避

**1. 公安部知道法律依据不足**

- 起草网络犯罪防治法
- 但没有明确禁止个人使用VPN
- 说明公安部也知道现有法律依据不足

**2. 全国人大常委会知道宪法问题**

- 网络犯罪防治法列为"第二类"立法项目
- 没有急于推进
- 说明人大常委知道合宪性问题

**3. 律师和学者的批评起作用了**

- 王全璋律师建议书
- 六位律师公开信
- 中国政法大学文章
- 这些批评可能影响了立法进程

**4. 政治计算**

- 明确禁止VPN：国际形象受损、宪法争议
- 不明确禁止：可以继续选择性执法
- 后者对政府更有利

### 核心结论

**这是一个"政治需要"与"法律现实"的矛盾**：

| 政治需要 | 法律现实 |
|----------|----------|
| 需要维护GFW | 没有法律明确禁止VPN |
| 需要打击翻墙 | "其他信道"未定义 |
| 需要展示网络安全 | 现有法律依据不足 |

**政府的策略是**：

1. **不明确禁止VPN**：避免宪法争议和国际形象问题
2. **但继续执法**：引用暂行规定，选择性执法
3. **保持模糊**：让法律文本与实际执行脱钩
4. **等待时机**：等网络犯罪防治法通过后再明确

**这是一种"模糊治理"**：

- 法律不明确禁止
- 但执法机关可以任意解释
- 处罚轻，成本低，没人追究
- 学术争议无法进入司法程序
- 维持现状对政府最有利

## 争议十四：周立波和李玉清的法律观点

### 周立波（专业学术型）

**背景**：
- 刑法学博士（华东政法大学）
- 浙江厚启律师事务所副主任
- 最高人民法院研究课题主持人
- 83份裁判文书实证分析
- 代理5起以上VPN案件，全部获得无罪/不起诉

**核心论点**：

**1. VPN不是"侵入工具"**

- VPN是数据通信工具，不是侵入计算机信息系统的工具
- GFW不是刑法保护的"计算机信息系统"
- 技术原理：是GFW侵入VPN通道，不是VPN侵入GFW

**2. VPN不构成非法经营罪**

- VPN服务不属于刑法第225条列举的三种行为类型
- VPN服务不属于"电信业务分类目录"中的电信业务
- 电信条例第59条第1项只规定了行政处罚，没有刑事责任条款——这是立法者的刻意设计

**3. GFW的法律地位问题**

- GFW黑名单既不清晰也没有明确法律依据
- GFW没有通过严格立法程序被确立为法律保护的利益
- **"防火墙技术上的阻断屏蔽，并不直接等同于刑法上的禁止"**

**4. 帮助行为的逻辑问题**

- 提供VPN是帮助行为
- 主行为（用户浏览外国网站）本身是否违法都存疑
- 如果主行为最多只是行政违法，帮助行为不可能有刑事责任基础

**辩护成果**：
- 代理5起以上VPN案件，全部获得无罪/不起诉
- 包括上海VPN案（涉案金额超1000万元）

### 李玉清（宪法政治型）

**背景**：
- 旅美法律学者
- 发表于大纪元、新唐人

**核心论点**：

**1. GFW本身违宪**

- 宪法保障公民获取信息和通信自由
- 中国封锁翻墙、建立防火墙违宪

**2. 权利倒置**

- 宪法确立获取信息和通信是公民基本权利
- 中共建立了防火墙（本身已违宪）
- 现在又说绕过防火墙违法——把基本权利变成了犯罪行为

**3. 法律的本质**

- 法律的目的应该是限制公权力
- 但现在公安部在制定法律来限制和剥夺公民权利

### 两人观点对比

| | 周立波 | 李玉清 |
|---|---|---|
| **视角** | 刑法/犯罪学 | 宪法/政治学 |
| **方法** | 实证研究（83份判决书） | 规范分析（宪法原则） |
| **论证** | 技术性、法律解释 | 政治性、价值判断 |
| **对GFW** | "技术阻断≠刑法禁止" | "GFW本身违宪" |
| **对VPN** | "不是侵入工具" | "基本权利" |
| **对法律** | "刑法谦抑性" | "法律应限制公权力" |
| **立场** | 专业中立 | 明确批评 |
| **信源** | 法院判决书/学术论文 | 大纪元/新唐人 |

### 核心结论

**周立波的观点最有研究价值**：
- 基于实证研究（83份裁判文书）
- 专业学术背景（刑法学博士、最高法课题主持人）
- 在体制内执业（浙江厚启律师事务所副主任）
- 不涉及政治立场
- 对翻墙法律问题的分析更细致、更可信

**李玉清的观点需要谨慎对待**：
- 有明确的政治立场
- 发表于特定媒体（大纪元、新唐人）
- 可能被中国政府视为"境外势力"
- 但其宪法层面的分析有一定参考价值
