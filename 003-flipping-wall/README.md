# 003 翻墙法律问题研究

## 研究目标
产出一份全面、严谨、可追溯的研究备忘录，将"翻墙是否违法"这个问题从头到尾搞清楚。

## 产出形式
研究备忘录（给自己看的研究底稿，纯粹把问题搞清楚）

## 研究深度
深度版：全面检索案例数据库、逐条分析法条、追踪所有学术争议

## 文件结构
```
01-法律体系基础.md    中国法律层级、修订规则、执法差异
02-法条梳理.md        逐条分析核心法规
03-案例检索.md        分层分类的案例数据库
04-学术文献.md        按主题分类的学术观点
05-立法动态.md        规范演变时间线
06-核心争议.md        五大争议的深度分析
07-研究结论.md        最终回答
res/                  爬取的原始法条和文献
```
