# 05 立法动态：规范演变时间线

## 一、暂行规定生命周期时间线

### 第一阶段：制定（1996-1997）

| 时间 | 事件 | 关键内容 |
|------|------|----------|
| 1994年5月 | 中国第一个国际专线正式接入互联网 | 中国互联网元年 |
| 1996年1月23日 | 国务院第42次常务会议通过暂行规定 | 制定程序完成 |
| 1996年2月1日 | 暂行规定发布（国务院令第195号） | 总理李鹏签署 |
| 1996年 | 暂行规定"暂行"开始 | 无"日落条款"，无时间限制 |
| 1997年5月20日 | 暂行规定第一次修订（国务院令第218号） | 仅删除"邮电部"，核心条款不变 |

**1996年立法背景**：
- 中国互联网刚起步（1994年接入国际互联网）
- 用户极少，主要为科研机构和高校
- GFW尚未建立（1998年才开始建设）
- VPN翻墙技术尚未出现
- "其他信道"的原始含义：私拉海底光缆、卫星天线等物理接入方式

### 第二阶段：配套法规制定（1997-2000）

| 时间 | 事件 | 关键内容 |
|------|------|----------|
| 1997年12月16日 | 安全保护管理办法发布（公安部令第33号） | 第5条（禁止行为）、第20条（处罚） |
| 1998年2月13日 | 实施办法发布（国务院信息化工作领导小组） | **第3条第3款："国际出入口信道，是指国际联网所使用的物理信道"** |
| 1998年 | GFW开始建设 | 中国网络审查制度正式建立 |
| 2000年 | 暂行规定内容固化 | 加入"接入网络"概念，第10条要求用户"必须通过接入网络进行国际联网" |

### 第三阶段：长期未修订（2000-2017）

| 时间 | 事件 | 关键内容 |
|------|------|----------|
| 2000年 | 《立法法》颁布 | 确立法律保留原则，但未追溯审查暂行规定 |
| 2000年 | 全国人大《关于维护互联网安全的决定》 | 确立在网络空间适用法律的原则 |
| 2011年1月8日 | 安全保护管理办法修订 | 仅删除"邮电部"，核心条款不变 |
| 2013年 | 斯诺登事件 | 互联网与国家安全问题凸显 |
| 2015年 | 《立法法》修正 | 第9条明确授权期限不得超过五年（但暂行规定不属于授权立法） |

**这一阶段的特点**：
- 暂行规定长期未被关注
- 翻墙用户极少，执法案例罕见
- 国务院没有主动修订或转化的意愿
- 胡凌（2008年）指出："尽管叫做'暂行规定'，里面一些内容也早已过时，但国务院似乎没有进一步修正的意向"

### 第四阶段：翻墙执法兴起（2017-2020）

| 时间 | 事件 | 关键内容 |
|------|------|----------|
| 2017年6月1日 | 网络安全法施行 | 第27条成为新的执法路径 |
| 2017年 | 32号文发布（工信部） | 仅针对企业跨境经营，不针对个人 |
| 2018年11月1日 | 151号令施行 | 为翻墙执法提供新的法律依据 |
| 2018年12月28日 | **韶关朱云枫案** | 首例翻墙行政处罚案；蓝灯VPN；487次；罚款1000元 |
| 2019年1月4日 | 黄成城案 | 仅传唤无正式处罚 |
| 2019年1月7日 | 通力律师事务所发表文章 | 系统阐述翻墙违法原因 |
| 2019年1月8日 | 杨迅律师发表不同见解 | 提出"接入网络"解释路径 |
| 2019年2月 | **赖亮平案** | 第一起翻墙行政诉讼；法院回避核心法律问题 |
| 2019年5月 | 孙东洋案 | 协助翻墙被判提供侵入计算机系统工具罪 |
| 2019年6月 | 刘某东海案 | 处罚时效问题：2017年行为2019年处罚 |
| 2019年7月2日 | **蒋鹏案** | 依据安全保护管理办法第20条而非暂行规定第6条 |

| 2020年5月17日 | **杨某某案** | 情指中心实时监控；引发胡锡进公开反对 |
| 2020年7月28日 | 易某案/陈某案 | 同一天两案，说明规模化执法 |
| 2020年10月24日 | **张韬案** | 首例因浏览维基百科被处罚；浙江近70例 |

**这一阶段的特点**：
- 翻墙执法从2018年开始大规模出现
- 执法依据不统一（暂行规定第6条/第8条/第10条/安全保护管理办法第20条/网络安全法第27条）
- 处罚力度差异极大（警告 vs 5000元罚款）
- 公安机关开始使用技术手段实时监控翻墙行为

### 第五阶段：立法博弈（2021-2024）

| 时间 | 事件 | 关键内容 |
|------|------|----------|
| 2021年11月14日 | 数安条例征求意见稿发布 | 第41条建立"数据跨境安全网关"；禁止"穿透、绕过"；被视为"翻墙入刑"信号 |
| 2021年11月29日 | 美国之音报道 | 分析人士指出"墙"的合法化进程 |
| 2022年 | 学界对数安条例征求意见稿的批评 | 多位学者指出第41条违宪 |
| 2024年3月10日 | **暂行规定第二次修订**（国务院令第777号） | **仅删除"邮电部"用语，改为"国家"；核心条款不变** |
| 2024年8月30日 | 数安条例正式发布 | 第39条删除"数据跨境安全网关"、删除"穿透、绕过"、删除"境内流量不得路由至境外"；**仅禁止"提供"，不禁止"使用"** |
| 2024年8月7日 | **龚某某案** | 倒查4年；网安+国保联合查处；宁德网警"只要翻墙就违法"公告 |
| 2024年10月19日 | 程大鹏案 | 高额罚款5000元；用第8条而非第6条 |
| 2025年1月1日 | 数安条例施行 | 正式生效 |

**这一阶段的特点**：
- 数安条例征求意见稿引发"翻墙入刑"担忧
- 正式版大幅弱化，对翻墙的规制力度显著降低
- 暂行规定2024年修订仅删除"邮电部"，未将其转化为正式法规
- 执法力度加大（倒查4年、高额罚款）

### 第六阶段：合宪性审查探索（2025-至今）

| 时间 | 事件 | 关键内容 |
|------|------|----------|
| 2025年 | 学界对暂行规定的合宪性审查 | 多位学者论证宪法第40条的"法律保留"原则 |
| 2026年2月5日 | **王全璋律师提交建议书** | 指《网络犯罪防治法（征求意见稿）》"越权立法"，违反宪法和立法法 |
| 2026年5月15日 | 行政法规制定程序条例第二次修订 | 第39条要求"暂行"名称用于授权立法；第25条要求涉及合宪性问题需征求全国人大常委会意见 |

**这一阶段的特点**：
- 合宪性审查成为新的争议焦点
- 王全璋律师的建议书为翻墙执法的合宪性审查提供了实践案例
- 行政法规制定程序条例的修订为暂行规定的审查提供了制度依据

## 二、暂行规定"暂行"了27年

### 时间线

```
1996年2月1日  暂行规定发布
    ↓
1997年5月20日  第一次修订（仅删除"邮电部"）
    ↓
   27年未实质性修订
    ↓
2024年3月10日  第二次修订（仅删除"邮电部"）
    ↓
   至今仍为"暂行"
```

### 关键节点

| 时间 | 距今 | 事件 |
|------|------|------|
| 1996年 | 28年 | 暂行规定发布 |
| 1997年 | 27年 | 第一次修订 |
| 1998年 | 26年 | GFW开始建设 |
| 2000年 | 24年 | 《立法法》颁布 |
| 2017年 | 7年 | 网络安全法施行 |
| 2018年 | 6年 | 首例翻墙行政处罚案 |
| 2024年 | 当年 | 第二次修订；数安条例施行 |
| 2026年 | 当年 | 行政法规制定程序条例第二次修订 |

### 为什么能"暂行"27年？

| 原因 | 说明 |
|------|------|
| 没有"日落条款" | 暂行法规不会自动失效 |
| 国务院没有主动转化的意愿 | 未将其转化为正式法规 |
| 翻墙执法是2018年后才大规模出现的 | 暂行规定长期未被关注 |
| 2024年修订时仅删除"邮电部" | 未将其转化为正式法规 |

### 学者的研究

**陈寒非《试验性法治：暂行立法的游击战逻辑》（2024年）**：

> "我国暂行法存在一些难以克服的内在矛盾：一是虽名为'暂行''试行'却**无明确的时间条款**；二是暂行法的形式与效力之间存在确定性冲突……三是虽然在一定程度上赋予了暂行法试验性功能，但在实践中**缺乏明确有效的总结提升机制**，很多暂行法施行较长时间后未能转化为法律。"

来源：http://law.newdu.com/c/20240303/18279.html

**胡凌《一九九八年之前的中国互联网立法》（2008年）**：

> "尽管叫做'暂行规定'，里面一些内容也早已过时，但**国务院似乎没有进一步修正的意向**。"

来源：https://nodebe4.github.io/opinion/2008-03-07/胡凌-一九九八年之前的中国互联网立法/

## 四、立法空白：28年未修订的实质问题

### 1996年的现实

- 中国互联网刚起步（1994年接入国际互联网）
- 用户极少，主要为科研机构和高校
- GFW尚未建立（1998年才开始建设）
- VPN翻墙技术尚未出现

### 2024年的现实

- 中国网民超10亿
- 翻墙用户估计达9000万（Global WebIndex数据）
- GFW已运行20余年
- VPN/SSR等翻墙技术高度成熟

### 立法滞后的核心问题

1. **"其他信道"的含义**：1996年指的是物理信道，是否应扩张解释为包括VPN？
2. **"接入网络"的含义**：1996年的网络层级结构与今天差异巨大
3. **处罚对象**：1996年主要针对物理层面的违规，是否应适用于应用层的加密通讯？
4. **执法机关**：1996年仅赋予公安机关处罚权，网信部门、市场监管部门是否有权处罚？

## 五、"暂行"27年：一部被遗忘的法规

### 5.1 行政法规制定程序条例的规定

**《行政法规制定程序条例》（2026年5月15日第二次修订）第39条**：

> "行政法规的名称一般称'条例'，也可以称'规定'、'办法'等。**国务院根据全国人民代表大会及其常务委员会的授权决定制定的行政法规，称'暂行条例'或者'暂行规定'。**"

**这意味着**："暂行"名称应该用于**授权立法**——即全国人大授权国务院制定的行政法规。

### 5.2 暂行规定是否属于授权立法？

**1. 立法法第8条（法律保留事项）**

| 事项 | 是否属于法律保留 |
|------|------------------|
| 犯罪和刑罚 | ✅ 是 |
| 限制人身自由的强制措施和处罚 | ✅ 是 |
| 网络管理 | ❌ 否 |

**2. 暂行规定的制定背景**

- 1996年制定时，涉及的是网络管理事项
- 网络管理不属于立法法第8条规定的"法律保留"事项
- 国务院可以**自行制定**行政法规，不需要全国人大授权

**3. 结论**

| 事实 | 含义 |
|------|------|
| 暂行规定用了"暂行"名称 | 但**不属于授权立法** |
| 行政法规制定程序条例要求"暂行"用于授权立法 | 暂行规定的**名称不符合要求** |
| 暂行规定制定于1996年，早于立法法 | 当时没有明确的"暂行"使用规范 |

### 5.3 这揭示了什么？

暂行规定"暂行"了27年，揭示了中国立法的一个深层问题：

| 问题 | 表现 |
|------|------|
| "暂行"没有时间限制 | 无"日落条款"，可以无限期"暂行" |
| "暂行"与"正式"效力相同 | 暂行法规在司法适用上是正式法源 |
| 缺乏总结提升机制 | 很多暂行法未能转化为正式法律 |
| 名称与实质不符 | 用了"暂行"名称，但不属于授权立法 |

### 5.4 对翻墙执法的影响

| 事实 | 影响 |
|------|------|
| 暂行规定"暂行"了27年 | 说明这是一部**被遗忘的法规** |
| 核心条款从未实质性修订 | 1996年的规定适用于2024年的互联网 |
| 2018年后才因为翻墙执法被重新关注 | 执法部门利用一部过时的法规进行处罚 |
| 名称与实质不符 | 用了"暂行"名称，但不属于授权立法 |

**"暂行"了27年，恰恰说明这是一部被遗忘的法规，直到2018年后才因为翻墙执法而被重新关注。**

## 六、2026年4月专项整治

### 6.1 行动层级与范围

2026年4月起，中国针对翻墙行为的打击从零星执法升级为**全国性系统性整治**：

**中央层面统一部署**：
- 工信部召集三大运营商专题会议，议题为"加强跨境数据专线违规连接互联网管理"
- 中央网信办2026年4月2日发出通知，召开"深入学习习近平关于网络强国的重要思想理论研讨会"
- 网信办2026年4月8日在北京举行全国网络法治工作会议

**基础设施层面——"拔线潮"**：
- 2026年4月1日起，全国多地IDC机房接到执法指令
- 执行方式：物理拔线——直接拔网线、断电源、整机下架
- 影响范围：广东、上海、北京等多省IDC机房，部分香港中转节点也受波及
- 据报道，执法人员从2025年5月开始主动购买机场订阅获取节点IP列表，通过ISP通报追溯到托管机房，4月1日集中行动
- 4月15-22日达到峰值，约60-70%国内中转链路同时失效

**技术升级**：
- SNI黑名单从约4-6万域名扩至8-10万域名
- QUIC/UDP限速从电信主线扩展到联通骨干层
- 对Cloudflare、Vercel等中转友好域名做精细化阻断
- 机器学习分类识别（OpenVPN、WireGuard等标准VPN协议检出率接近100%）

### 6.2 各地执行文件

**江苏苏州**：
- 《关于配合运营商全省网络安全专项治理工作的告知函》
- 自2026年4月1日起开展"违规跨境访问"专项整治
- 要求接入服务商签署合规承诺书，对被通报IP立即停用
- 不设整改期限，直接断端口、终止服务

**陕西**：
- 《关于全面封禁海外流量及严禁翻墙业务的紧急通知》
- 自即日起，所有IP地址全面禁止访问中国大陆以外地址（含港澳台）
- 严禁承载VPN/代理等翻墙业务
- 违规立即封禁IP/停机，资源不保留，费用不退还

### 6.3 《网络犯罪防治法（征求意见稿）》

**时间**：2026年1月31日，公安部发布

**关键条款**：
- 第44条直接将翻墙工具和服务列为打击目标
- 拟定处罚力度：
  - 个人：最高罚款50万元人民币 + 拘留15天
  - 机构：最高500万元 + 吊销营业执照
  - "提供技术支持或协助"也纳入打击——分享翻墙教学、推荐工具都可能触法

**王全璋律师的回应**：
- 2026年2月5日向全国人大常委会提交建议书
- 指《网络犯罪防治法（征求意见稿）》"越权立法"，违反宪法和立法法
- 核心论点：涉及限制人身自由的处罚只能由法律来设定，公安部无权制定

### 6.4 VPN服务商影响

- 国际翻墙软件"快连VPN"（LetsVPN）于2026年4月28日宣布终止中国大陆业务运营
- 消息人士称，中共高层已将VPN定义为"颠覆国家政权的工具"

## 七、未来可能的立法走向

### 短期（1-2年）
- 《网络犯罪防治法》可能正式出台，大幅加重翻墙处罚
- 暂行规定可能被《网络犯罪防治法》取代或补充
- 2026年专项整治可能常态化

### 中期（3-5年）
- 可能出台专门的网络审查法规
- 可能明确GFW的法律地位
- 可能统一翻墙处罚标准

### 长期
- 习近平总书记关于网络的表态：强调**网络安全**，建设**网络强国**
- 翻墙执法不太可能放松，反而可能进一步加强
- 2026年专项整治已显示从个人罚款转向基础设施关停的趋势
