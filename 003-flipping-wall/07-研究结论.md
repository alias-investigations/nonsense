# 07 研究结论：翻墙是否违法？

## 一、核心结论

### 个人翻墙行为
**不构成犯罪，是否构成行政违法存在重大争议。**

理由：
1. **VPN不属于"其他信道"**：实施办法第3条将"国际出入口信道"定义为"物理信道"，VPN是虚拟加密隧道，不属于物理信道。暂行规定第6条禁止的是"使用其他信道"，VPN不在禁止之列。
2. **翻墙不违反"必须通过接入网络"**：翻墙用户已通过接入网络入网，翻墙只是在已入网的基础上访问特定境外网站。
3. **翻墙不构成任何犯罪**：不构成提供侵入非法控制工具罪（VPN不是专门用于侵入的工具），不构成非法经营罪（翻墙服务不属于任何电信业务类型），不构成拒不履行信息网络安全管理义务罪（需满足特定条件）。
4. **数安条例正式版不禁止个人使用**：征求意见稿第41条曾被视为"翻墙入刑"信号，但正式版第39条仅禁止"提供"，不禁止"使用"。
5. **暂行规定可能违反宪法第40条**：宪法第40条要求限制通信自由必须由"法律"规定，暂行规定只是行政法规，可能违反"法律保留"原则。
6. **暂行规定"暂行"了27年**：行政法规制定程序条例第39条要求"暂行"名称用于授权立法，但暂行规定不属于授权立法（网络管理不属于法律保留事项）。名称与实质不符，是一部被遗忘的法规。

### 提供翻墙工具/服务
**不构成犯罪，但可能构成行政违法。**

理由：
1. **不构成提供侵入非法控制工具罪**：VPN具有广泛合法用途，不是"专门用于"侵入的工具。
2. **不构成非法经营罪**：翻墙服务不属于任何电信业务类型，不存在合法经营的市场。
3. **可能违反暂行规定第10条**：提供翻墙服务过程中，租用境外代理服务器的行为可能未通过接入网络。
4. **可能违反数安条例第39条**：提供破坏、避开技术措施的程序、工具。

## 二、现有执法的问题

### 问题一：法律适用错误
三条主要执法路径均存在法律适用错误：
- **暂行规定第6条**：VPN不是"其他信道"
- **暂行规定第10条**：翻墙不是"未通过接入网络"
- **网络安全法第27条**：VPN不是"专门用于危害网络安全活动的程序、工具"

### 问题二：法理共识与执法实践的脱节

**核心发现**：律师界关于"VPN≠其他信道"的法理共识在实践中完全无效。

| 环节 | 状态 |
|------|------|
| 律师界 | 基本一致：VPN≠其他信道 |
| 学术界 | 没有专门研究（CSSCI期刊上几乎无讨论） |
| 行政执法 | 直接定性，不论证 |
| 司法判决 | 完全回避（不采纳、不否定、不讨论） |

**这意味着**：法律解释不取决于"法理正确与否"，而取决于"执法机关如何认定"。一旦行政机关形成"既定结论"，法院不会挑战。

### 问题三：执法标准不统一
- 不同地区公安机关对相同行为作出不同定性
- 处罚依据不统一（第6条/第10条/网络安全法第27条）
- 处罚金额差异极大（200元到105万元）

### 问题三：选择性执法
- 翻墙用户估计达9000万
- 被处罚者极少
- 执法具有选择性

### 问题四：立法滞后
- 暂行规定制定于1996年，28年来未实质性修订
- 条文中的"信道""接入网络"等概念针对的是物理层面的违规行为
- 与VPN翻墙技术的实际场景脱节

### 问题五：名称与实质不符
- **行政法规制定程序条例第39条**要求"暂行"名称用于**授权立法**
- 但暂行规定**不属于授权立法**（网络管理不属于法律保留事项）
- 国务院可以自行制定行政法规，不需要全国人大授权
- **"暂行"27年**：无"日落条款"，可以无限期"暂行"；缺乏总结提升机制；被遗忘的法规直到2018年后才因翻墙执法被重新关注

### 问题六：合宪性存疑
- **暂行规定可能违反宪法第40条**：宪法第40条要求限制通信自由必须由"法律"规定，暂行规定只是行政法规，可能违反"法律保留"原则
- **行政法规制定程序条例的要求**：征求意见稿要求行政法规"符合立法技术规范"，正式版本要求涉及合宪性问题需征求全国人大常委会意见
- **学者背书**：王全璋律师、张翔教授、全国人大常委会法工委等都论证了宪法第40条的"法律保留"原则，但无人直接论证暂行规定第6条违反宪法第40条

**张翔教授的宪法学分析（2021+2023）**：
1. **通信权的保护范围**：通话记录当然落入宪法第40条通信权的保护范围（《通信权的宪法释义与审查框架》，2021年）
2. **"检查通信"是示例性规定**：如果将"检查通信"理解为"示例性规定"，则宪法第40条容有对通信权限制的其他可能性——但仍需满足法律保留要求
3. **通信权的双重法律保留结构**：加重法律保留（第40条第2句：通信检查三要件）+ 单纯法律保留（第40条第1句：通信自由和秘密）
4. **法律保留的中国方案**（《基本权利限制法律保留的中国方案》，2023年）：中国宪法基本权利章的独特结构决定了不能照搬德国"单纯—加重—无"三层结构；第51条排除了"无法律保留"阶层存在的可能性；"单纯—加重"的两层结构同时存在于人身自由、财产权、通信权等权利的内部

## 三、核心法条的正确理解

### 暂行规定第6条
> 任何单位和个人不得自行建立或者使用其他信道进行国际联网。

**正确理解**：
- "其他信道"指"其他物理信道"（参照实施办法第3条第3款）
- VPN是虚拟加密隧道，不是物理信道
- VPN用户的数据仍然通过运营商的物理信道传输
- **结论：VPN不属于"其他信道"，第6条不适用于VPN翻墙**

### 暂行规定第10条
> 个人、法人和其他组织使用的计算机或者计算机信息网络，需要进行国际联网的，必须通过接入网络进行国际联网。

**正确理解**：
- "接入网络"是网络层级结构中的一个环节（用户→接入网络→互联网络→国际出入口信道）
- 翻墙用户已通过接入网络入网
- 翻墙是在已入网的基础上访问特定境外网站
- **结论：翻墙不违反"必须通过接入网络"，第10条不适用于VPN翻墙**

### 数安条例第39条
> 国家采取措施，防范、处置网络数据跨境安全风险和威胁。任何个人、组织不得提供专门用于破坏、避开技术措施的程序、工具等...

**正确理解**：
- 仅禁止"提供"行为，不禁止"使用"行为
- "破坏、避开技术措施"是否涵盖VPN存疑
- **结论：个人使用VPN翻墙不违反第39条**

## 四、立法建议

### 短期
1. **统一执法标准**：明确翻墙行为的法律定性，避免同案不同判
2. **限制处罚范围**：明确翻墙行为最多构成行政违法，不构成犯罪
3. **规范处罚程序**：要求公安机关在处罚翻墙行为时进行充分的法律论证

### 中期
1. **修订暂行规定**：明确"其他信道"的含义，将其限定为物理信道
2. **修订实施办法**：更新"国际出入口信道"的定义，适应技术发展
3. **出台专门规定**：明确翻墙行为的法律地位和处罚标准
4. **合宪性审查**：由全国人大常委会对暂行规定第6条进行合宪性审查，明确其是否违反宪法第40条

### 长期
1. **逐步放宽网络审查**：随着对外开放，逐步降低GFW的封锁范围
2. **建立合法翻墙渠道**：为科研、商务等合法需求提供合规的跨境网络接入途径
3. **完善个人信息保护**：在保障国家安全的前提下，保护公民的通信自由和隐私权

---

## 五、对翻墙用户的建议

1. **了解法律风险**：虽然翻墙不构成犯罪，但可能存在行政违法风险
2. **避免敏感行为**：翻墙后避免从事可能违法的活动（如传播违法信息）
3. **保留证据**：如被处罚，保留处罚决定书等证据，以便后续救济
4. **依法救济**：如认为处罚不当，可依法申请行政复议或提起行政诉讼

---

## 六、对提供翻墙工具者的建议

1. **了解法律风险**：提供翻墙工具可能构成行政违法
2. **避免商业化运营**：大规模商业化运营增加法律风险
3. **合规经营**：如从事跨境网络服务，应依法取得相关许可
4. **法律咨询**：在从事相关业务前，咨询专业律师
