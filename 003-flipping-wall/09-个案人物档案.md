# 09 个案人物档案：具体的人、机构、背景

## 研究方法
本章采用"侦探式"研究路径，从抽象法律回到具体的人和事。重点考察：
- 当事人身份背景
- 执法人员/机构
- 判决背景
- 地方财政/政治环境

## 案件1：黄成城（重庆荣昌，2019.1.4）

### 当事人
- **姓名**：黄成城
- **出生**：1983年，中专学历
- **身份**：重庆荣昌区居民，政治敏感人物（不同政见者）
- **前科**：
  - 2004年因网上发表文章被判"煽动颠覆国家政权罪"3年（2009年4月3日获释）
  - 2011年因"茉莉花革命"网帖被劳教2年（后被撤销，获国家赔偿10.4万元）
  - 2013年3月24日与等了7年的女友陈玲结婚

### 案件背景
- 2018年11月：好友刘刚（荣昌区异议人士）被国保传唤、查扣手机
- 国保在调查刘刚过程中关联到黄成城
- 国保问黄成城："你觉得刘刚像不像精神病人？"
- 国保说黄成城"不老实！"

### 处罚过程
- 2019年1月4日：以"擅自建立、使用非法定信道进行国际联网"传唤8小时
- 警方检查通讯设备，将其境外网站政治言论打印出来让其确认
- 警方指称：翻墙登陆境外网站是违法行为，情节严重的可以判刑
- 警方警告：以后不得再翻墙，如不听警告，下次可能从重处理或判刑
- **没有正式行政处罚**，仅传唤+恐吓

### 执法机构
- **单位**：重庆荣昌区公安局
- **主导部门**：国保（危害国家安全犯罪侦查）
- **执法人员**：未公开

### 时间背景与触发因素
- **2018年11月**：好友刘刚（荣昌区异议人士）被国保传唤、查扣手机
- **2019年1月4日**：黄成城被传唤——刘刚案调查的延伸
- **国保在调查刘刚过程中关联到黄成城**
- **触发因素**：政治关联——翻墙是打击政治异见的法律工具，与净网行动无关

### 关键特征
**政治案件，翻墙是借口**。警方真正目标是他的政治言论，翻墙只是找到的法律工具。没有正式行政处罚决定书，说明警方也知道翻墙处罚缺乏依据，只是用作恐吓手段。时间上与刘刚案高度关联（刘刚2018.11被传唤→黄成城2019.1.4被传唤）。

## 案件2：朱云枫（韶关南雄，2018.12.28）

### 当事人
- **姓名**：朱云枫
- **年龄**：30岁（2018年时）
- **身份**：广东韶关南雄市居民，普通居民（非政治人物）

### 案件背景
- 2018年8月-12月：手机安装蓝灯（Lantern Pro），连接自家宽带翻墙
- 被发现方式：**处罚决定书未说明**（技术手段？举报？）

### 处罚过程
- 处罚日期：2018年12月28日
- 处罚依据：暂行规定第6条+第14条
- 处罚结果：警告+罚款1000元
- 案号：韶雄公（网）行罚决字〔2019〕1号
- 文书编号：PCS4402201812280000000080167110

### 执法机构
- **单位**：南雄市公安局
- **承办部门**：公共信息网络安全监察大队（网监大队）
- **关键信息**：这是该大队2019年第1张行政处罚决定书
- **公开方式**：广东公安执法信息公开平台

### 舆论影响
- 2019年1月4日由韶关市公安局在广东公安执法信息公开平台公开
- 引发全国媒体关注（BBC、RFA、DW、大纪元等）
- 网友发现：违法吸毒罚款500元，翻墙罚款1000元——"翻墙比吸毒严重？"

### 时间背景与触发因素
- **151号令刚生效2个月**：公安部《公安机关互联网安全监督检查规定》2018年11月1日生效，赋予警方远程检测权力（第16条）
- **同日网信办约谈28家应用商店**：2018年12月28日，国家网信办约谈28家应用商店，要求下架违规App
- **南雄网监大队2019年第1张处罚决定书**：说明这是新年后首批执法
- **触发因素**：新规实施后的执法测试——151号令赋予新权力，需要通过实际案例验证执法路径

### 关键特征
**首例公开的个人翻墙处罚案**。之前只有提供翻墙工具的处罚，此案首次将处罚对象扩展到个人用户。开创了"翻墙=使用非法定信道"的执法先例。151号令生效后仅2个月，可能是首批"测试性执法"之一。

- 2019年5月26-27日：编造虚假信息在推特散布，到庐丰政府起哄闹事

- **开庭日期**：2019年11月28日（公开开庭）
- **判决日期**：2019年12月24日
- **判决结果**：驳回诉讼请求（维持处罚）
- **判决依据**：行政诉讼法第69条

### 出庭人员
| 角色 | 姓名 | 职务 |
|------|------|------|
| 原告 | 赖亮平 | 自行诉讼（无律师） |
| 被告上杭县公安局出庭负责人 | 汤春康 | 副局长 |
| 被告上杭县公安局代理人 | 傅文辉、张正锰 | — |
| 被告上杭县政府出庭负责人 | 丁焱志 | — |
| 被告上杭县政府代理人 | 陈亮金 | — |

### 法院认定
1. 关于重复处罚：两次处罚依据不同（第6条 vs 治安管理处罚法第26条），不属于重复处罚
2. 关于处罚时限：推特言论在被查获时仍发布在平台上，违法行为持续存在，未超出处罚时限
3. 关于翻墙处罚合法性：**法院没有审查**，只审查了程序问题

### 执法机构
- **单位**：上杭县公安局
- **承办部门**：庐丰派出所
- **庐丰畲族乡**：省级生态乡镇，上杭县第一个中共农村党支部诞生地
- **上杭县公安局局长**：林占俊（副县长兼）
- **上杭县政府**：县长王波

### 时间背景与触发因素
- **2019年5月26-27日**：在推特发布"人贩子"虚假信息，到庐丰政府起哄闹事
- **2019年6月**：被传唤处罚——从造谣到处罚仅约2-3周
- **触发因素**：寻衅滋事——翻墙是附带处罚，主要处罚是治安管理处罚法第26条（寻衅滋事拘留10日）

### 关键特征
**唯一一例行政诉讼，但法院没有审查翻墙处罚本身的合法性**。法院只审查了程序问题（是否重复处罚、是否超出时限），没有审查"VPN是否属于其他信道"这个核心法律问题。说明法院在回避翻墙案件的实体审查。翻墙处罚（警告）与寻衅滋事处罚（拘留10日）力度悬殊，说明翻墙只是"搭便车"。

### 警方检测能力
- 长春案揭示：**警方拥有实时检测VPN连接的技术能力**
- 检测到VPN连接 → 37分钟内发送短信要求到案
- 这解释了朱云枫"一周487次"如何被精确统计
- 承德程序员的翻墙行为很可能也是被技术手段发现

### 核心发现
1. **大多数翻墙案件背后有其他驱动力**：政治言论、寻衅滋事、其他违法犯罪。翻墙本身很少成为独立的执法目标。
2. **翻墙是"口袋罪"式的法律工具**：警方可以用翻墙来处罚几乎任何人，因为数千万人都在翻墙。
3. **地方财政压力可能驱动高额罚没**：承德案尤为明显。
4. **法院回避实体审查**：赖亮平案中，法院没有审查翻墙处罚本身的合法性。
5. **执法高度选择性**：数千万翻墙用户，被处罚者极少。

## 触发因素分析（修正版）

### 净网行动相关性质疑
早期分析曾假设翻墙执法与"净网行动"直接相关。但用户正确指出：净网行动每年持续数月，案件与净网时间间隔达~10个月，因果关系过于宽泛。**需要更精确的触发因素分析。**

### 修正后的多因素触发模型

翻墙执法由**多重因素叠加触发**，而非单一"净网行动"：

| 触发因素 | 案例 | 作用机制 |
|----------|------|----------|
| **新规实施** | 朱云枫案（151号令2018.11.1生效） | 新规赋予新执法权力，需测试 |
| **政治关联** | 黄成城案（刘刚案延伸）、承德程序员案 | 翻墙是打击政治异见的法律工具 |
| **地方维稳** | 赖亮平案（寻衅滋事） | 翻墙是寻衅滋事的附带处罚 |
| **财政压力** | 承德程序员案 | 罚没收入成为地方财政来源 |
| **技术升级** | 长春案、2026年专项整治 | 实时检测能力提升→执法效率提高 |
| **运动式执法** | 净网期间案件集中 | 地方公安有完成指标的压力 |

### 逐案触发因素详解

**案件1：黄成城（2019.1.4）**
- 触发因素：**政治关联**
- 时间线：刘刚2018.11被传唤 → 国保调查中关联黄成城 → 2019.1.4被传唤
- 与净网无关，纯属政治案件的延伸

**案件2：朱云枫（2018.12.28）**
- 触发因素：**新规实施（151号令）+ 运动式执法**
- 时间线：151号令2018.11.1生效 → 2个月后朱云枫被处罚 → 同日网信办约谈28家应用商店
- 151号令赋予警方远程检测权力，需要通过实际案例验证执法路径
- 可能是首批"测试性执法"之一

**案件3：赖亮平（2019.6）**
- 触发因素：**地方维稳（寻衅滋事）**
- 时间线：2019.5.26-27推特造谣+庐丰政府起哄 → 约2-3周后被处罚
- 翻墙处罚（警告）与寻衅滋事处罚（拘留10日）力度悬殊
- 翻墙只是"搭便车"

**案件4：承德程序员（2023.8）**
- 触发因素：**政治嫌疑 → 财政压力**
- 时间线：2022.9安全部门上门（怀疑Twitter）→ 后查明无关 → 翻墙被发现 → 2023.8下达处罚（~11个月）
- 政治嫌疑引发调查，翻墙被发现，财政压力驱动高额罚没
- 承德罚没收入占比8%（河北平均3.7%），税收下降19.9%

**案件5：长春案（2024.11.10）**
- 触发因素：**技术检测驱动**
- 时间线：2024.9长春网安宣传周 → 2024.11.10实时检测发现VPN连接 → 37分钟内响应
- 警方实时监控系统主动发现，非举报或运动式执法

**案件6：蕲春刑事案（2024）**
- 触发因素：**提供工具牟利**
- 刑事打击对象是"提供工具"，不是"使用工具"
- 与个人翻墙处罚性质不同

**案件7：东安案（2023.8）**
- 触发因素：**其他违法犯罪**
- 翻墙只是入口，真正处罚的是其他违法行为
- "目的、动机不同，同时涉嫌其他违法犯罪"

### 核心结论
1. **净网行动不是翻墙执法的直接触发因素**：时间间隔太宽泛（~10个月），净网是持续性运动，无法解释具体案件的触发
2. **翻墙执法的真实动机是多元的**：政治关联、新规测试、地方维稳、财政压力、技术升级
3. **技术能力是关键变量**：151号令（2018）→ 裁量基准（2016/2020）→ 实时检测（2024）→ 专项整治（2026）
4. **地方差异显著**：承德（财政压力）、荣昌（政治）、南雄（新规测试）、上杭（维稳）——同一部法律，不同地方有不同执法逻辑
