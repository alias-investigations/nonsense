# 案例深度分析：廖某湖北咸宁翻墙炒币案

## 一、案件事实

### 当事人
- 湖北省咸宁市居民
- 廖某（男性）
- 团队成员：朱某、胡某、郑某、孙某（共4人）
- 另有2人涉案（总计7人被行政处罚）

### 时间线
| 时间 | 事件 |
|------|------|
| 2019年12月5日 | 廖某开始通过VPN翻墙炒币 |
| 2024年8月 | 公安机关查获该工作室 |
| 2024年12月17日 | 上海律师吴恩祥披露行政处罚决定书 |
| 2024年12月18日 | 多家媒体跟进报道 |
| 2024年12月19日 | 曼昆律师事务所刘红林律师发表评论 |

### 处罚决定
- **处罚类型**：警告+罚款1.5万元（法定上限）
- **没收违法所得**：9万元
- **收缴工具**：8部手机+8台电脑
- **处罚依据**：暂行规定第6条+第14条
- **处罚机关**：咸宁市公安局

### 关键细节
1. **翻墙炒币**：在境外交易APP上低价买入、高价卖出虚拟货币（OTC交易）
2. **经营规模**：8部手机+8台电脑，4人团队，运营约5年
3. **群众举报**：系群众举报后被查获
4. **"犯罪团队"措辞**：公安机关文章中一度使用"犯罪团队"措辞，但最终仅作出行政处罚

---

## 二、执法特点分析

### 2.1 违法所得的认定

**公安机关的认定**：
- 将虚拟货币交易获利9万元全部认定为"违法所得"
- 理由：这些获利是通过翻墙（违法行为）获得的

**问题**：
- 炒币本身不违法
- 翻墙与炒币获利之间是否有法律上的因果关系？
- 如果一个人翻墙后进行合法交易，交易获利是否属于"违法所得"？

### 2.2 处罚力度

| 项目 | 金额/数量 |
|------|-----------|
| 罚款 | 1.5万元（法定上限） |
| 没收违法所得 | 9万元 |
| 收缴工具 | 8部手机+8台电脑 |
| **总计** | **10.5万元+全部设备** |

**分析**：
- 罚款已达法定上限
- 全部获利被没收（5年经营收益归零）
- 所有设备被收缴

### 2.3 与其他案例的对比

| 案例 | 翻墙行为 | 处罚结果 |
|------|----------|----------|
| 廖某案 | 翻墙炒币（OTC交易） | 罚款1.5万+没收9万 |
| 张韬案 | 访问维基百科 | 当场训诫 |
| 承德程序员案 | 远程工作 | 罚款200元+没收105万 |

## 三、法律分析

### 3.1 违法所得的法律定义

**暂行规定第14条**：
> "违反本规定第六条、第八条和第十条的规定的，由公安机关责令停止联网，给予警告，可以并处15000元以下的罚款；有违法所得的，没收违法所得。"

**问题**：
1. "违法所得"是指翻墙行为本身的收益，还是翻墙后所有行为的收益？
2. 如果翻墙后进行合法交易，交易获利是否属于"违法所得"？
3. 翻墙与炒币获利之间是否有法律上的因果关系？

### 3.2 律师观点

**上海律师吴恩祥的观点**：
- 要将炒币获利认定为"违法所得"，必须证明翻墙与获利行为的高度关联性
- 炒币本身不违法，仅仅因为使用VPN翻墙，就将其后续合法行为所获得的收益认定为违法所得，逻辑存在问题
- **违法所得应限于翻墙行为本身直接带来的收益**（如通过出售VPN服务盈利）

**曼昆律师事务所刘红林律师的观点**：
- 个人"翻墙"在目前内地的执法实践中是"玄学"
- "翻墙"与"炒币"在法理和逻辑上被强行绑定并不严谨
- 该案被冠以"全国第一案"的标题过于危言耸听

### 3.3 虚拟货币交易的合法性

**"9.24通知"**（《关于进一步防范和处置虚拟货币交易炒作风险的通知》）：
- 仅要求风险自担
- 未禁止个人投资虚拟货币

**律师普遍观点**：
- 现行中国内地不允许虚拟货币交易所在内地展业
- 但法律并未明确禁止内地居民合法合规地交易和持有虚拟货币

## 四、案件意义

### 执法层面
1. **违法所得认定的扩大化**：将翻墙后合法交易的获利认定为"违法所得"
2. **处罚力度**：罚款1.5万元（法定上限）+没收全部获利
3. **"翻墙炒币"成为新的执法重点**

### 研究层面
1. **违法所得的认定标准**：翻墙行为本身的收益 vs 翻墙后所有行为的收益
2. **翻墙与后续行为的因果关系**：手段违法是否导致所有后续行为违法？
3. **法律依据的滞后性**：1996年的暂行规定如何适用于2024年的虚拟货币交易？
