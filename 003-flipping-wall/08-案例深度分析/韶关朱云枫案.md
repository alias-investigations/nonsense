# 案例深度分析：韶关朱云枫案——首例公开翻墙处罚案

## 一、案件事实

### 基本信息
- **案号**：韶雄公（网）行罚决字〔2019〕1号
- **文书编号**：PCS4402201812280000000080167110
- **当事人**：朱云枫，30岁，广东韶关南雄市
- **承办单位**：南雄市公安局公共信息网络安全监察大队
- **处罚日期**：2018年12月28日

### 违法事实
- 2018年8月至12月期间
- 在手机上安装蓝灯（Lantern Pro）翻墙软件
- 连接自家宽带网络翻墙上网
- 被查处前一周内登录487次

### 处罚依据
- 暂行规定第6条："计算机信息网络直接进行国际联网，必须使用邮电部国家公用电信网提供的国际出入口信道。任何单位和个人不得自行建立或者使用其他信道进行国际联网。"
- 暂行规定第14条：违反第6条的处罚

### 处罚结果
- 警告
- 罚款1000元

### 同日事件
- 重庆荣昌区也出现类似案件
- 黄成城被传唤，收到《被传唤人家属通知书》
- 同样指控"擅自建立、使用非法定信道进行国际联网"

---

## 二、本案的特殊意义

### 首例公开
- 这是第一起引起广泛舆论关注的个人翻墙行政处罚案
- 之前只有提供翻墙工具的处罚案例
- 此案首次将处罚对象扩展到**个人用户**

### 执法逻辑的起点
- 此案确立了执法机关的基本论证模式：
  1. VPN = 其他信道
  2. 使用VPN = 使用其他信道进行国际联网
  3. 违反暂行规定第6条
- 这一模式被后续案例广泛复制

### 法律争论的起点
- 通力律师事务所2019年1月发表两篇文章，形成了"两条路线"的争论
- 一条路线认同第6条的适用（VPN=其他信道）
- 另一条路线认为应适用第10条（未通过接入网络）

---

## 三、执法机关的论证模式

### 模式一：暂行规定第6条（多数派）
**核心逻辑**：
```
VPN = 其他信道
使用VPN = 使用其他信道进行国际联网
→ 违反暂行规定第6条
→ 适用第14条处罚
```

**特点**：
- 将"其他信道"扩大解释为包括VPN
- 不区分"建立"和"使用"
- 简单直接，容易操作

### 模式二：暂行规定第10条（少数派）
**核心逻辑**：
```
翻墙用户没有通过接入网络访问境外网站
→ 违反暂行规定第10条"必须通过接入网络进行国际联网"
→ 适用第14条处罚
```

**特点**：
- 不直接挑战"其他信道=物理信道"的解释
- 换一个角度论证翻墙违法
- 在执法实践中较少采用

### 两种模式的矛盾
- 同一个翻墙行为，被认定为违反不同的条款
- 第6条说的是"信道"问题，第10条说的是"网络"问题
- 两个条款针对的是不同的违法行为
- 执法机关内部对翻墙行为的法律定性存在分歧

---

## 四、学界对两种模式的反驳

### 对模式一的反驳（VPN≠其他信道）

**王宇扬（2020）**：
- "国际出入口信道"在实施办法第3条中被定义为"物理信道"
- "其他信道"应与"国际出入口信道"同属物理层面
- VPN是虚拟加密隧道，不是物理信道
- VPN用户的数据仍然通过运营商的物理信道传输
- 立法者不可能预见到VPN翻墙技术

**程俊峰等（2026）**：
- 如果确有建立非法定信道来翻墙的，应明确指出其建立的非法定信道是何种形式
- 不能顾名思义地将代理访问的路径直接联想为非法定信道
- 翻墙的基本原理是代理访问，是互联网常见应用服务

### 对模式二的反驳（翻墙≠未通过接入网络）

**王宇扬（2020）**：
- 翻墙不是"不通过公用电信网提供国际出入口信道的其他方式"
- 翻墙也不构成"接入网络"行为
- Art.10（2000版）不涵盖翻墙

**程俊峰等（2026）**：
- 翻墙的过程，从家庭网络到ISP机房，再到骨干网，直到越过国际出入口
- 和所有其他正常上网行为的接入方式和路径都一样
- 没有、也没必要非法接入网络
- 唯一的区别是到达境外后通过代理服务器间接访问被墙的网站

## 五、本案暴露的执法问题

### 问题一：选择性执法的起点
- 2018年之前，几乎没有人因个人翻墙被处罚
- 此案开创了先例
- 但执法仍然是高度选择性的——数千万翻墙用户，被处罚者极少

### 问题二：法律解释的随意性
- 执法机关没有对"VPN为什么是其他信道"进行论证
- 直接将VPN等同于"其他信道"
- 这是一种望文生义的解释方法

### 问题三：立法与执法的脱节
- 暂行规定制定于1996年
- 立法者不可能预见到VPN翻墙技术
- 执法机关将1996年的概念扩张解释为适用于2018年的新技术

## 六、执法依据的源头：重庆2016年裁量基准

### 发现
翻墙执法的制度起点不是韶关案，而是**重庆2016年裁量基准**。

### 关键文件
- **文件名称**：《重庆市公安机关网络安全管理行政处罚裁量基准》
- **文号**：渝公发〔2016〕105号
- **发布日期**：2016年7月27日
- **修订日期**：2020年2月5日（2020年版增加了对个人翻墙的处罚标准）

### 2016年版完整内容（渝公发〔2016〕105号）

**总则**：
- 制定依据：行政处罚法、计算机信息系统安全保护条例、互联网上网服务营业场所管理条例、暂行规定、安全保护管理办法等
- 适用范围：本市各级公安机关对违反网络安全管理的违法行为作出行政处罚决定
- 处罚原则：以事实为依据，与违法行为的性质、情节以及社会危害程度相适应

**第六至十二条（国际联网相关违法行为）**：

裁量基准将以下七种行为合并规定了统一的处罚标准：
1. 擅自建立、使用非法定信道进行国际联网（第6条）
2. 接入网络未通过互联网络接入国际联网（第8条）
3. 未经许可从事国际联网经营业务
4. 未经批准擅自进行国际联网
5. 未通过接入网络进行国际联网（第10条）
6. 未经接入单位同意接入接入网络
7. 未办理登记手续接入接入网络

**法律依据**：暂行规定第6条、第8条、第10条、第14条

**2016年版处罚标准**：
| 情形 | 处罚 |
|------|------|
| 不以盈利为目的，初次 | 责令停止联网+警告 |
| 以盈利为目的，违法所得≤5000元 | 责令停止联网+警告+没收违法所得 |
| 以盈利为目的，违法所得5000-10000元 | 责令停止联网+警告+个人≤5000元/单位5000-10000元罚款+没收 |
| 以盈利为目的，违法所得≥10000元或责令停止联网后继续实施 | 责令停止联网+警告+10000-15000元罚款+没收 |

**2020年修订版（渝公规〔2020〕1号）**：
- 2020年2月5日印发，2020年3月1日实施
- 通过重庆市司法局备案（渝文审〔2020〕252号）
- 吸收了其他省市经验，明确了32种常见违法行为
- 对《网络安全法》中公安机关予以处罚的18个违法行为名称进行了明确
- 处罚标准与2016年版基本一致

**政策解读（2023-03-30发布）**：
重庆市公安局官方解读：
> "《裁量基准》旨在规范重庆市公安机关行使网络安全管理行政处罚裁量权，促进行政执法公平公正，维护行政相对人合法权益。"

### 关键发现

1. **裁量基准没有定义"非法定信道"**——直接照搬暂行规定条文，没有解释VPN为什么属于"非法定信道"
2. **裁量基准没有定义"接入网络"**——同样直接照搬条文
3. **裁量基准将第6条和第10条合并处罚**——这意味着执法机关内部已经将两种解释路径混同适用
4. **裁量基准是公安机关内部文件**——不是法律、法规、规章，只是执法参考
5. **裁量基准的法律效力存疑**——能否作为处罚依据？

### 2017年媒体反应

**联合早报（2017-04-16）**：
- 重庆市公安出台的处罚裁量基准文件中，并没有出现"翻墙"两字
- 该字眼是记者对文件内容的个人解读和概括
- 复旦大学网络空间治理研究中心副主任沈逸：单凭该处罚裁量基准文件总结出重庆当局将对个人"翻墙"行为进行惩罚，是过于简单化的误读
- 沈逸：文件中提到的"擅自使用非法定信道"不等同于"个人翻墙"

**DW（2017-03-28）**：
- 重庆处罚基准配合工信部清理互联网，把条文规定得细致和具体化
- 南华早报：清理的对象已扩大到绕开政府防火墙的个人用户
- 同重庆市2011年出台的类似规定相比，老规定只惩罚供应商，不惩罚个人用户

**RFA（2017-03-28）**：
- 重庆的这项互联网安全规定去年7月就已开始生效、但日前才被公布
- 大赦国际研究人士：该规定的措辞如此含糊不清，存在被用于任何企业或个人的可能性

### 执法传导路径（完整版）

## 七、更多执法案例

### 张韬舟山案（2020）
- **案号**：舟定公(解)行罚决字[2020]01692号
- **事实**：2019年上半年至2020年10月期间，通过百度搜索到翻墙软件"LANTERN"并进行下载，多次利用"LANTERN"翻墙软件非法访问维基百科网站查询资料
- **处罚**：当场训诫
- **意义**：说明执法力度差异巨大——从训诫到没收105万

### 蕲春刑事案（2024）
- **案号**：（2024）鄂1126刑初400号
- **事实**：李某宇搭建VPN售卖网站，以8-200元套餐提供翻墙服务；肖某杰经李某宇传授搭建4个VPN售卖网站
- **判决**：提供侵入、非法控制计算机信息系统程序、工具罪
  - 李某宇：有期徒刑三年，缓刑五年，罚金5万，追缴违法所得61.2万
  - 肖某杰：有期徒刑三年，缓刑三年，罚金1万，追缴违法所得3.4万
- **意义**：提供翻墙工具可能构成刑事犯罪（区别于个人使用）

### 炒币翻墙案（2024）
- **事实**：4人以挂VPN方式擅自使用非法定信道进行国际联网，通过境外网站与客户联系，从事虚拟货币交易
- **处罚**：廖某责令停止联网+警告+罚款15000元+没收违法所得9万元+收缴手机8部、电脑8套
- **法律依据**：暂行规定第6条+第14条，治安管理处罚法第11条
- **意义**：翻墙+其他违法活动（虚拟货币交易）=没收违法所得

### 东安连续查处案（2023）
- **事实**：东安县公安局连续查处蒋某、张某、王某三人擅自使用VPN翻墙软件进行国际联网
- **法律依据**：暂行规定第6条+第14条，实施办法第7条+第22条
- **意义**：三人"目的、动机不同，同时涉嫌其他违法犯罪"——说明翻墙本身只是入口，真正处罚的是其他违法行为

## 八、"487次"数据来源调查

### 已知事实
- 处罚决定书原文："被查处前一周内登录487次"
- 处罚机关：南雄市公安局**公共信息网络安全监察大队**（网监大队）
- 没有任何公开来源解释这个数字是怎么统计出来的

### 已找到的相关线索

**1. 处罚决定书本身不解释数据来源**
- 决定书只陈述"登陆次数为487次"，没有说明数据来自ISP日志、DPI系统还是其他来源
- 月光博客、BBC、中国数字空间等媒体转述了这句话，但都没有追问数据来源

**2. 通用VPN检测技术（非本案具体证据）**
- DPI（深度包检测）：在ISP骨干网识别VPN协议签名
- ISP连接日志：记录VPN服务器IP、连接时间、协议类型、数据量
- 商业监控系统：傲盾"跨境VPN监控系统"可"还原和存储加密VPN流量的元数据，呈现一次VPN代理的完整过程"
- 反诈APP：RFA报道（2026-04-07）称"公安系统就会显示出来"——但这是2026年的技术，2018年未必有
- 2025年专利CN121691088A：福建紫讯的VPN检测方法——但2018年不存在

**3. 蓝灯(Lantern)的技术特点**
- 蓝灯是P2P架构，不是传统VPN
- 动态协议切换（AnyTLS、Hysteria 2、VLess等）
- 蓝灯声称"No Logs"政策（美国非营利组织）
- P2P架构和协议切换会导致频繁断线重连

**4. 相关但非直接证据**
- 2024年福建龚某某案：2020年翻墙，2024年才被处罚——说明ISP日志留存时间很长
- 公安部第三研究所展示过可监控Telegram的工具——但时间不详
- 南京公司展示过可自动检测翻墙行为的工具——但时间不详

### 未找到的证据
- 原始处罚决定书截图（@watchwci原帖已不可访问）
- 南雄市公安局关于检测方法的任何公开说明
- 任何技术报告将"487次"与具体检测方法联系起来
- 尚权律师事务所分析文章（URL失效）

### 推断（待验证）
- "登录"最可能指VPN连接会话（不是网页访问次数）
- 数据最可能来自ISP连接日志或DPI系统记录
- 网监大队有能力获取ISP日志和DPI数据
- 但**没有直接证据证明本案中数据的具体来源**
